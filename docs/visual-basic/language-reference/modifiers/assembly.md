# <a name="assembly-visual-basic"></a>Assembly(Visual Basic)
소스 파일의 시작 부분에 있는 특성이 전체 어셈블리에 적용 되도록 지정 합니다.  
  
## <a name="remarks"></a>주의  
 많은 특성 클래스 또는 속성과 같은 개별 프로그래밍 요소와 관련이 있습니다. 꺾쇠 괄호 안에 특성 블록을 연결 하 여 특성을 적용 (`< >`), 선언문에 직접.  
  
 소스 파일의 시작 부분에는 특성 블록을 배치 하 고 있는 특성으로 식별 특성에 다음 요소 뿐 아니라 전체 어셈블리에 관련 된 경우는 `Assembly` 키워드입니다. 현재 어셈블리 모듈에 적용 되는 경우 사용 된 [모듈](../../../visual-basic/language-reference/modifiers/module-keyword.md) 키워드입니다.  
  
 적용할 수 있습니다도 특성 AssemblyInfo.vb 파일에 있는 어셈블리를 주 소스 코드 파일에 특성 블록을 사용할 필요가 없습니다 있는 경우.
