# <a name="notinheritable-visual-basic"></a>NotInheritable(Visual Basic)
클래스를 기본 클래스로 사용할 수 없음을 지정 합니다.  
  
## <a name="remarks"></a>설명  
 **용어를 대체**합니다. 상속할 수 없는 클래스 라고도 *봉인* 클래스입니다.  
  
 `NotInheritable` 한정자는이 컨텍스트에서 사용할 수 있습니다.
