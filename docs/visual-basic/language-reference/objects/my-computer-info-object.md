# <a name="mycomputerinfo-object"></a>My.Computer.Info 개체

## <a name="remarks"></a>설명  
 속성에 대 한 내용은 `My.Computer.Info` 개체, 참조 <xref:Microsoft.VisualBasic.Devices.ComputerInfo>합니다.
