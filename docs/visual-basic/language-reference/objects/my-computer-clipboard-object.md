# My.Computer.Clipboard Object
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

클립보드 조작을 위한 메서드를 제공합니다.  
  
## 설명  
 메서드 및 속성에 대 한 자세한 내용은 `My.Computer.Clipboard` 개체을 참조 하십시오 <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>.  
  
 자세한 내용은 [Storing Data to and Reading from the Clipboard](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md)를 참조하십시오.  
  
> [!NOTE]
>  메서드를 사용할 수도 있습니다 해당 <xref:System.Windows.Forms.Clipboard?displayProperty=fullName> 클립보드를 조작할 수 있는 클래스입니다.
