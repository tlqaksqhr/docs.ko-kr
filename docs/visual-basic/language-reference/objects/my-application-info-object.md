# <a name="myapplicationinfo-object"></a>My.Application.Info 개체
버전 번호, 설명, 로드 된 어셈블리 등 응용 프로그램에 대 한 정보를 가져오기 위한 속성을 제공 합니다.  
  
## <a name="remarks"></a>설명  
 메서드 및 속성에 대 한 자세한 합니다 `My.Application.Info` 개체를 참조 하십시오 <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>합니다.  
  
> [!NOTE]
>  속성을 사용할 수는 <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> 디스크의 파일에 대 한 정보는 클래스입니다.

 **클래스:** <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>
