# <a name="expression-does-not-produce-a-value"></a>식이 값을 산출하지 않습니다.
같은 값을 산출 컨텍스트에 값을 산출 하지 않는 되는 식을 사용 하려는 `Sub` 컨텍스트에서 여기서는 `Function` 가 필요 합니다.  
  
 **오류 ID:** BC30491  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   값을 생성 하는 식을 변경 합니다.
