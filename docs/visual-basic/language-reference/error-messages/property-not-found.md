# <a name="property-not-found"></a>속성을 찾을 수 없습니다.
이 개체는 지정된 된 속성을 지원 하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  속성의 이름의 철자를 확인 합니다.  
  
2.  개체가 실제로 "캡션" 또는 비슷한 이름의 속성을 지원 하는 경우는 "text" 속성 처럼 것에 액세스 하려고 시도 하지 않는 고유 하도록 개체의 설명서를 확인 합니다.
