# Need property array index
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

이 속성 값은 단일 값이 아닌 배열로 구성됩니다.  액세스하려는 속성 배열에 대해 인덱스를 지정하지 않았습니다.  
  
### 이 오류를 해결하려면  
  
-   구성 요소 설명서를 참조하여 배열에 적합한 인덱스의 범위를 찾습니다.  속성 액세스 문에서 적절한 인덱스를 지정합니다.
