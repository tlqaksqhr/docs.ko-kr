# Property array index is not valid
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

속성 배열 인덱스에 잘못된 값을 지정했습니다.  
  
### 이 오류를 해결하려면  
  
-   구성 요소의 설명서를 확인하여 현재 인덱스가 지정된 속성에 대해 올바른 범위에 있는지 확인합니다.
