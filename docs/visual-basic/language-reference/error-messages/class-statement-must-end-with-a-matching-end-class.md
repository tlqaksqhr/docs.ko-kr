# <a name="39class39-statement-must-end-with-a-matching-39end-class39"></a>&#39; 클래스 &#39; 문은 짝이 되는 &#39;로 끝나야 합니다. End 클래스 &#39;
`Class`시작 하는 데 사용 되는 `Class` 차단; 되므로 일치 하는 블록의 시작 부분에만 나타날 수 `End Class` 문 블록을 종료 합니다. 중복가 있거나 `Class` 문이 종료 한 프로그램 `Class` 블록와 함께 `End Class`합니다.  
  
 **오류 ID:** BC30481  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제거할 때 불필요 한 `Class` 문.  
  
-   결론을 내릴는 `Class` 일치 하는 블록 `End Class`합니다.
