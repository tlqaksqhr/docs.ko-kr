# Unable to get serial port names because of an internal system error
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`My.Computer.Ports.SerialPortNames` 속성을 호출할 때 내부 오류가 발생했습니다.  
  
### 이 오류를 해결하려면  
  
1.  [디버거 기본 사항](/visual-studio/debugger/debugger-basics)에서 자세한 문제 해결 정보를 확인합니다.  
  
2.  오류가 발생한 상황을 파악하여 Microsoft 기술 지원 서비스에 문의합니다.
