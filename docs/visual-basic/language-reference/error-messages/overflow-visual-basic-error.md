# <a name="overflow-visual-basic-error"></a>오버플로가 발생했습니다(Visual Basic 오류).
리터럴 할당 하 고 데이터 형식의 범위를 벗어나는 값을 나타냅니다.  
  
 **오류 ID:** BC30036  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   입력 하 고 해당 범위에 맞게 리터럴을 다시 작성 하는 데이터 대상에 대 한 값 범위를 참조 하십시오.
