# Statement is not valid in a namespace
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

네임스페이스 수준에서는 문을 사용할 수 없습니다.  모듈, 인터페이스, 클래스, 대리자, 열거형 및 구조체만 선언할 수 있습니다.  
  
 **오류 ID:** BC30001  
  
### 이 오류를 해결하려면  
  
-   문을 모듈, 클래스, 인터페이스, 구조체, 열거형 또는 대리자 정의 안으로 이동합니다.
