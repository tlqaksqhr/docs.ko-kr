# <a name="device-io-error"></a>장치 입/출력(I/O) 오류입니다.
사용자 프로그램에서 프린터 또는 디스크 드라이브와 같은 장치를 사용 하는 동안 입력 또는 출력 오류가 발생 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   장치가 제대로 작동 하는지 확인 하 고 작업을 다시 시도 합니다.
