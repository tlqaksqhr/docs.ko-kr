# XML axis properties do not support late binding
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

형식화되지 않은 개체에 대해 XML 축 속성을 참조했습니다.  
  
 **오류 ID:** BC31168  
  
### 이 오류를 해결하려면  
  
-   XML 축 속성을 참조하기 전에 개체가 강력한 형식의 <xref:System.Xml.Linq.XElement> 개체인지 확인합니다.
