# Method does not have a signature compatible with the delegate
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

사용하려는 메서드와 대리자의 시그니처가 호환되지 않습니다.  `Delegate` 문은 대리자 클래스의 매개 변수 형식과 반환 형식을 정의합니다.  이 대리자 형식의 인스턴스를 만들기 위해 호환되는 매개 변수 형식과 반환 형식이 일치하는 모든 프로시저를 사용할 수 있습니다.  
  
 **오류 ID**: BC36563
