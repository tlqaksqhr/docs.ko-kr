# <a name="modules-visual-basic"></a>모듈(Visual Basic)
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]여러 모듈을 사용 하면 문자열 조작, 시스템 정보 가져오기, 파일 및 디렉터리 작업을 수행 수학 계산 수행을 비롯 하 여 코드에서 일반적인 작업을 간소화 하 고 등을 제공 합니다. 다음 표에서에서 제공 되는 모듈 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]합니다.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.Constants>|기타 상수를 포함합니다. 코드에서 아무 곳 이나 이러한 상수를 사용할 수 있습니다.|  
|<xref:Microsoft.VisualBasic.ControlChars>|인쇄 및 표시 텍스트에 대 한 상수 제어 문자를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.Conversion>|숫자를 문자열, 문자열을 숫자로, 하나의 데이터 형식을 다른 형식으로 다른 밑으로 10 진수 변환 하는 멤버가 포함 되어 있습니다.|  
|<xref:Microsoft.VisualBasic.DateAndTime>|현재 날짜 또는 시간, 날짜를 계산, 하거나 반환 하는 날짜 또는 시간, 날짜 또는 시간을 설정 시간 프로세스의 기간 멤버가 포함 되어 있습니다.|  
|<xref:Microsoft.VisualBasic.ErrObject>|런타임 오류 및 발생 하거나 오류를 해결 하는 방법에 대 한 정보가 포함 됩니다.|  
|<xref:Microsoft.VisualBasic.FileSystem>|파일, 디렉터리 또는 폴더 및 시스템 작업을 수행 하는 멤버를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.Financial>|재무 계산을 수행 하는 데 사용 되는 절차가 포함 됩니다.|  
|<xref:Microsoft.VisualBasic.Globals>|현재 스크립팅 엔진 버전에 대 한 정보를 포함합니다.|  
|<xref:Microsoft.VisualBasic.Information>|반환, 테스트 또는 배열 크기, 형식 이름 등의 정보를 확인 하는 멤버를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.Interaction>|포함 된 개체, 응용 프로그램 및 시스템과 상호 작용 하는 멤버입니다.|  
|<xref:Microsoft.VisualBasic.Strings>|문자열 작업, 문자열의 길이 가져오는 문자열 검색 문자열 서식 다시 지정 등을 수행 하는 멤버를 포함 합니다.|  
|<xref:Microsoft.VisualBasic.VBMath>|포함 된 멤버 수치 연산을 수행 합니다.|
