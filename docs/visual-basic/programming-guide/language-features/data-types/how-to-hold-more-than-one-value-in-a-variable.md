# <a name="how-to-hold-more-than-one-value-in-a-variable-visual-basic"></a>방법: 변수에 두 개 이상의 값 사용(Visual Basic)
변수에 둘 이상의 값으로 선언 하는 경우는 *복합 데이터 형식을*합니다.  
  
 [복합 데이터 형식](../../../../visual-basic/programming-guide/language-features/data-types/composite-data-types.md) 구조, 배열 및 클래스를 포함 합니다. 복합 데이터 형식의 변수에 기본 데이터 형식 및 기타 복합 형식의 조합을 포함할 수 있습니다. 구조체와 클래스 코드 뿐만 아니라 데이터를 저장할 수 있습니다.  
  
### <a name="to-hold-more-than-one-value-in-a-variable"></a>변수에 두 개 이상의 값을  
  
1.  변수를 사용 하려면 복합 데이터 형식을 결정 합니다.  
  
2.  복합 데이터 형식이 아직 정의 되지 않은 정의 변수에서 사용할 수 있도록 합니다.  
  
    -   구조체를 정의 [Structure 문을](../../../../visual-basic/language-reference/statements/structure-statement.md)합니다.  
  
    -   정의 된 배열을 [Dim 문](../../../../visual-basic/language-reference/statements/dim-statement.md)합니다.  
  
    -   클래스를 정의 [Class 문](../../../../visual-basic/language-reference/statements/class-statement.md)합니다.  
  
3.  사용 하 여 변수를 선언는 `Dim` 문.  
  
4.  변수 이름 뒤에 `As` 절.  
  
5.  에 따라는 `As` 키워드와 적절 한 복합 데이터 형식의 이름입니다.
