# <a name="type-conversions-in-visual-basic"></a>Visual Basic의 형식 변환
하나의 데이터 형식에서 다른 형식으로 값을 변경 하는 프로세스 라고 *변환*합니다. 변환은 하나 *확대* 하거나 *축소*데이터 용량입니다. 관련 된 형식에 따라 합니다. 또한 *암시적* 하거나 *명시적*소스 코드 구문에에서 따라 합니다.
