# 이름이 같은 다른 멤버가 &#39;Shadows&#39;로 선언되었으므로 &lt;typename&gt; &#39;&lt;membername&gt;&#39;은 &#39;Shadows&#39;로 선언해야 합니다.
이 속성 또는 프로시저의 이름이 `Shadows` 한정자로 표시된 다른 속성 또는 프로시저의 이름과 같습니다.  
  
 **오류 ID:** BC30695  
  
### 이 오류를 해결하려면  
  
-   `Shadows` 한정자를 사용합니다.
