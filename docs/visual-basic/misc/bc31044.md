# &#39;As&#39; 절로 선언된 이벤트에는 대리자 형식을 지정해야 합니다.
대리자가 아닌 형식을 지정했습니다.  
  
 **오류 ID:** BC31044  
  
### 이 오류를 해결하려면  
  
1.  `As` 절을 제거합니다.  
  
2.  `As` 절을 사용하여 대리자를 지정합니다.
