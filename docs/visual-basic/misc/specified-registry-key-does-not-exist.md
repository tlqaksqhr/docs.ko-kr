# <a name="specified-registry-key-does-not-exist"></a>지정한 레지스트리 키가 없습니다.
지정된 레지스트리 키가 존재하지 않는 경우  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제공하는 레지스트리 키에서 경로가 올바르게 지정되었는지 확인합니다.  
  
-   쓰기를 시도하기 전에 레지스트리 키를 만듭니다.
