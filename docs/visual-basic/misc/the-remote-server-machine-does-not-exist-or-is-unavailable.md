# 원격 서버 컴퓨터가 없거나 사용할 수 없습니다(Visual Basic).
사용할 수 없거나 존재하지 않는 원격 서버에 연결하려고 합니다. 네트워크 연결 문제로 인해 이 오류가 발생할 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   네트워크에 다시 연결하여 시도합니다.
