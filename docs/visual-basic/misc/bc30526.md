# <a name="property-39ltpropertynamegt39-is-39readonly39"></a>속성 &#39; &lt;propertyname&gt;&#39;는 &#39; 읽기 전용 &#39;
`ReadOnly`로 선언된 속성에서 값을 할당하려고 했습니다.  
  
 **오류 ID:** BC30526  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   속성 선언에서 `ReadOnly` 지정자를 제거합니다.
