# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39; 사용자 지정 이벤트 &#39; 일치 하는 &#39;로 끝나야 합니다. 종료 이벤트 &#39;
`Custom Event` 선언은 `End Event` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31114  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Custom Event` 선언이 `End Event` 문으로 끝나는지 확인합니다.
