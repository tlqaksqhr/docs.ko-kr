# 대상 버전의 .NET Compact Framework에서 런타임에 바인딩 오버로드 확인을 지원하지 않습니다.
대상 버전의 .NET Compact Framework에서 런타임에 바인딩 오버로드 확인을 지원하지 않습니다.  
  
 **오류 ID:** BC30764  
  
### 이 오류를 해결하려면  
  
-   형식 개체의 매개 변수를 적절한 형식으로 캐스팅합니다.
