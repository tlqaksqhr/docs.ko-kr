# <a name="white-space-may-not-appear-here"></a>공백은 여기에 표시할 수 없습니다.
공백이 XML 요소 리터럴 이름에 포함되었습니다.  
  
 **오류 ID:** BC31177  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 요소 리터럴의 이름에서 공백을 제거합니다.
