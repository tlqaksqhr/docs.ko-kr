# <a name="badly-formed-xml-file-39ltcommentfilegt39-cannot-be-included"></a>XML의 형식이 잘못되었습니다. 파일 &#39; &lt;commentfile&gt; &#39; 포함할 수 없습니다.
XML 주석 중 하나의 `<include>` 태그가 존재하지 않는 파일이나 잘못된 형식의 XML이 포함된 파일을 가리킵니다.  
  
 **오류 ID:** BC42321  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  파일이 있는지 확인합니다.  
  
2.  파일의 XML이 유효한지 확인합니다.
