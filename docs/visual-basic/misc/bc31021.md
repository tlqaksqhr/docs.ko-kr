# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic 컴파일러에서 다음과 같은 오류를 복구할 수 없으면: &lt;오류&gt;
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 컴파일러는 다음 오류를 복구할 수 없습니다: `error`>.  
  
 일부 IDE 기능을 사용하지 않도록 설정했습니다.  
  
 내부 오류가 발생 했습니다는 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] 컴파일러입니다.  
  
 **오류 ID:** BC31021  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  작업을 저장하고 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]를 다시 시작합니다.  
  
2.  그래도 오류가 다시 발생하면 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]을 다시 설치합니다.  
  
3.  다시 설치 후에도 오류가 계속 발생하면 Microsoft 기술 지원 서비스에 알립니다.
