# 속성 접근자를 &#39;&lt;keyword&gt;&#39;로 선언할 수 없습니다.
`Get` 또는 `Set` 프로시저 선언에 속성 프로시저에 적합하지 않은 키워드가 포함되어 있습니다.  
  
 [Get Statement](../../visual-basic/language-reference/statements/get-statement.md) 및 [Set Statement](../../visual-basic/language-reference/statements/set-statement.md)에서는 액세스 한정자\(`Public`, `Protected`, `Friend`, `Protected Friend`, `Private`\)만 허용됩니다.  
  
 **오류 ID:** BC31099  
  
### 이 오류를 해결하려면  
  
-   `Get` 또는 `Set` 문에서 잘못된 키워드를 제거합니다.
