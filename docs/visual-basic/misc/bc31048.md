# <a name="properties-with-no-required-parameters-cannot-be-declared-39default39"></a>필수 매개 변수가 없는 속성을 선언할 수 없습니다 &#39;기본값&#39;
필수 매개 변수는 없지만 `Default` 한정자는 있는 속성을 지정했습니다.  
  
 **오류 ID:** BC31048  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   기본 속성에 대한 필수 매개 변수를 선언합니다.  
  
-   속성 정의에서 `Default` 한정자를 제거합니다.
