# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39;Namespace&#39; 문은 짝이 되는 끝나야 &#39;끝 Namespace&#39;
`Namespace` 블록은 `End Namespace` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC30626  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `End Namespace` 블록 끝에 `Namespace` 문을 추가합니다.
