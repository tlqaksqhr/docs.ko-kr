# <a name="type-argument-39lttypeargumentnamegt39-does-not-satisfy-the-39structure39-constraint-for-type-parameter-39lttypeparameternamegt39"></a>형식 인수 &#39; &lt;되었으며&gt;&#39; 충족 하지 않습니다는 &#39; 구조 &#39; 형식 매개 변수 &#39;에 대 한 제약 조건 &lt;typeparametername&gt;&#39;
제네릭 형식에 제공되는 형식 인수가 해당 형식 매개 변수에서 값 형식 제약 조건을 충족하지 않습니다.  
  
 제약 조건 목록은 형식 매개 변수에 전달되는 형식 인수에 대해 요구 사항을 적용합니다. 제약 조건 목록에 특정 클래스나 인터페이스를 포함하지 않는 경우 다음 중 하나를 지정하여 일반 요구 사항을 적용할 수 있습니다.  
  
-   형식 인수는 값 형식( [구조체(Visual Basic)](http://msdn.microsoft.com/en-us/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) 제약 조건 포함)이어야 합니다.  
  
-   형식 인수는 참조 형식( [클래스(Visual Basic)](http://msdn.microsoft.com/en-us/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) 제약 조건 포함)이어야 합니다.  
  
 동일한 형식 매개 변수에 `Structure` 와 `Class` 를 둘 다 지정할 수 없으며 두 번 이상 지정할 수도 없습니다.  
  
 **오류 ID:** BC32105  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   값 형식의 형식 인수를 선택합니다.
