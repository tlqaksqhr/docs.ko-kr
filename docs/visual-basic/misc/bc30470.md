# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39;MyClass&#39; 클래스 외부에 사용할 수 없습니다
클래스 구조 외부에서 `MyClass` 키워드를 사용했습니다.  
  
 **오류 ID:** BC30470  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `MyClass` 선언을 클래스 구조로 이동합니다.
