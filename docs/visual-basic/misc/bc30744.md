# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>참조 된 배열 요소에 값으로 &#39; 아무것도 &#39;
참조되는 배열 요소가 설정되지 않았거나 이전에 `Nothing`으로 설정되었습니다.  
  
 **오류 ID:** BC30744  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   배열의 값이 올바르게 설정되고 있는지 확인합니다.
