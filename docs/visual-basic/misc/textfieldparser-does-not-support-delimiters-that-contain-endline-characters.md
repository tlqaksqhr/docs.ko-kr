# TextFieldParser는 EndLine 문자가 포함된 구분 기호를 지원하지 않습니다.
EndLine 문자를 포함하는 구분 기호가 제공되었지만 `TextFieldParse`r는 EndLine 문자가 포함된 구분 기호를 지원하지 않습니다.  
  
### 이 오류를 해결하려면  
  
-   올바른 구분 기호를 제공합니다.
