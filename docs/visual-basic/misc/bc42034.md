# 파일 이름이 이미 다른 GUID와 체크섬 값으로 선언되어 있음
파일을 선언하려고 했지만 다른 GUID 및 체크섬 값을 갖는 파일이 이미 있습니다.  
  
 **오류 ID:** BC42034  
  
### 이 오류를 해결하려면  
  
-   파일 이름 및 경로가 모두 정확한지 확인합니다.  
  
-   필요하지 않은 경우 중복된 파일을 삭제합니다.
