# <a name="query-expressions-are-not-valid-in-debug-windows"></a>디버그 창에서 쿼리 식은 올바르지 않습니다.
디버깅 세션 중 LINQ 쿼리가 **조사식** 또는 **직접 실행** 창에 입력되었습니다. 디버그 창에서는 쿼리를 사용할 수 없습니다.  
  
 **오류 ID:** BC36609
