# &#39;&lt;function&gt;&#39;이 선언되지 않았습니다.
'\<function\>'이 선언되지 않았습니다. 함수는 Microsoft.VisualBasic 네임스페이스로 이동되었습니다.  
  
 **오류 ID:** BC30818  
  
### 이 오류를 해결하려면  
  
-   `Microsoft.VisualBasic` 네임스페이스에서 적절한 함수를 사용합니다.
