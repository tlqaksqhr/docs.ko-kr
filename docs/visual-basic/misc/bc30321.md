# <a name="39case39-cannot-follow-a-39case-else39-in-the-same-39select39-statement"></a>&#39; 대/소문자 &#39; 따를 수 없는 &#39; 다른 사례 &#39; 동일한 &#39; Select&#39; 문
`Case Else` 문은 최초 `Case`와 일치하는 부분이 없는 경우 실행될 문을 지정합니다. 동일한 `Case` 블록에서 `Case Else` 문이 `Select` 다음에 있습니다.  
  
 **오류 ID:** BC30321  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Case Else` 를 `Case` 문 다음에 적절한 위치로 이동합니다.
