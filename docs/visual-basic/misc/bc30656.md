# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>필드 '&lt;fieldname&gt;'는 지원 되지 않는 형식
필드를 나타낼 수 없는 형식으로 선언 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]합니다.  
  
 **오류 ID:** BC30656  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  필드를 지원되는 형식으로 정의합니다.  
  
2.  필드의 데이터를 설명하는 새 형식을 개발합니다.
