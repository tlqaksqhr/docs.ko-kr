# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;오류&gt;: &#39; &lt;structurename1&gt; &#39; 포함 &#39; &lt;structurename2&gt;&#39;
구조체가 자신을 해당 멤버 중 하나로 지정하는 경우 순환 구조 정의가 발생합니다.  
  
 **오류 ID:** BC30293  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   구조체 멤버의 이름을 변경합니다.
