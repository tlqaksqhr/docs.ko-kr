# &#39;Finally&#39;는 &#39;Try&#39; 문에서 한 번만 사용할 수 있습니다.
`Finally`는 `Try...Catch...Finally` 블록을 완료하는 데 사용되므로 블록의 끝에 한 번만 나타날 수 있습니다.  
  
 **오류 ID:** BC30381  
  
### 이 오류를 해결하려면  
  
1.  중복 `Finally`를 찾아서 제거합니다.
