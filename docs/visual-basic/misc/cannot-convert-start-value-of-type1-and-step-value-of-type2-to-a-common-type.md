# <a name="cannot-convert-start-value-of-type-39lttype1gt39-and-step-value-of-type-39lttype2gt39-to-a-common-type"></a>형식 &#39;의 시작 값으로 변환할 수 없습니다. &lt;type1&gt;&#39; 및 단계 값 형식 &#39;&lt; 유형 2&gt;&#39; 공용 형식
지정된 `For...Next` 컨트롤 값의 형식이 서로 달라서 공통 형식으로 변환할 수 없으므로 유효하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  모든 값이 공통 형식으로 변환될 수 있도록 하나 이상의 값 형식을 변경합니다.
