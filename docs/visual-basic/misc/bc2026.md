# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>경고 번호 '&lt;번호&gt;'옵션'에 대 한&lt;optionname&gt;' 구성할 수 없는 되었거나 유효 하지 않습니다
경고 ID가 잘못되었습니다. ID는 0보다 큰 정수 값이어야 합니다.  
  
 **오류 ID:** BC2026  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제공된 경고 ID를 검사하여 유효한지 확인합니다.
