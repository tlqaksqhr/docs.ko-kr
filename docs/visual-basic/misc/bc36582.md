# &#39;&lt;typename&gt;&#39;에 정의된 확장 메서드 &#39;&lt;methodname&gt;&#39;의 인수가 너무 많습니다.
확장 메서드 호출에서 제공된 인수 개수가 메서드에 대해 정의된 인수 개수를 초과합니다.  
  
 **오류 ID:** BC36582  
  
### 이 오류를 해결하려면  
  
-   확장 메서드의 정의를 참조하고 호출에서 불필요한 인수를 제거합니다.
