# <a name="too-many-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>확장 메서드는 인수가 너무 많습니다 &#39; &lt;methodname&gt; &#39; 에 정의 된 &#39; &lt;typename&gt;&#39;
확장 메서드 호출에서 제공된 인수 개수가 메서드에 대해 정의된 인수 개수를 초과합니다.
