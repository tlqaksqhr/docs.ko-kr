# <a name="39notinheritable39-classes-cannot-have-members-declared-39ltspecifiernamegt39"></a>&#39; NotInheritable &#39; 클래스는 선언 된 멤버 &#39; 포함할 수 없습니다. &lt;specifiername&gt;&#39;
재정의 한정자는 해당 멤버를 재정의할 수 없기 때문에 `NotInheritable` 클래스와 함께 사용할 수 없습니다.  
  
 **오류 ID:** BC30607  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Overridable`, `NotOverridable`또는 `MustOverride`와 같은 재정의 한정자를 클래스 정의에서 제거합니다.
