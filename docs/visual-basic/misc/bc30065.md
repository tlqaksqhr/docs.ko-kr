# Function 또는 Property에서는 &#39;Exit Sub&#39;를 사용할 수 없습니다.
`Exit Sub`는 `Function` 프로시저 또는 `Property` 프로시저에 나타납니다.`Exit` 문은 해당 문이 발생한 블록과 일치해야 합니다.  
  
 **오류 ID:** BC30065  
  
### 이 오류를 해결하려면  
  
-   `Exit Sub`를 `Exit Function` 또는 `Exit Property` 문으로 적절하게 바꿉니다.
