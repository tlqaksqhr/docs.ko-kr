# 직접 실행 창에서는 &#39;Using&#39; 문을 사용할 수 없습니다.
`Using` 문은 소스 코드에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30186  
  
### 이 오류를 해결하려면  
  
-   **즉시 실행** 창에서 `Using` 문을 제거합니다.
