# &#39;&lt;membername&gt;&#39; 멤버가 &#39;&lt;classname&gt;&#39; 클래스에 없습니다.
'\<membername\>' 멤버가 '\<classname\>' 클래스에 없습니다. 대개 'Microsoft.VisualBasic.dll'이 일치하지 않으면 이러한 상황이 발생합니다.  
  
 정의된 멤버를 찾을 수 없습니다.  
  
 **오류 ID:** BC31097  
  
### 이 오류를 해결하려면  
  
1.  프로그램을 다시 컴파일하여 오류가 다시 발생하는지 확인합니다.  
  
2.  오류가 다시 발생하면 작업을 저장하고 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)]를 다시 시작합니다.  
  
3.  그래도 오류가 계속 발생하면 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)]을 다시 설치합니다.  
  
4.  다시 설치 후에도 오류가 계속 발생하면 Microsoft 기술 지원 서비스에 알립니다.
