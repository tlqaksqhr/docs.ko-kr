# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39; RemoveHandler &#39; 선언 일치 하는 &#39;로 끝나야 합니다. RemoveHandler &#39; 종료
`RemoveHandler` 선언은 `End RemoveHandler` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31116  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `RemoveHandler` 선언이 `End RemoveHandler` 문으로 끝나는지 확인합니다.
