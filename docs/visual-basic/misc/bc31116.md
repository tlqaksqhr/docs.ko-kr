# <a name="39removehandler39-declaration-must-end-with-a-matching-39end-removehandler39"></a>&#39;RemoveHandler&#39; 선언은 짝이 되는로 끝나야 &#39;끝 RemoveHandler&#39;
`RemoveHandler` 선언은 `End RemoveHandler` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31116  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `RemoveHandler` 선언이 `End RemoveHandler` 문으로 끝나는지 확인합니다.
