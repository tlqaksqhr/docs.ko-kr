# <a name="39end-raiseevent39-must-be-preceded-by-a-matching-39raiseevent39-declaration"></a>&#39; RaiseEvent &#39; 종료 뒤에 야 일치 하는 &#39; RaiseEvent &#39; 선언
`End RaiseEvent` 문이 해당 `RaiseEvent` 문 없이 발생합니다. `End RaiseEvent` 는 해당 `RaiseEvent` 문 뒤에 와야 합니다.  
  
 **오류 ID:** BC31126  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   앞의 `RaiseEvent` 문이 유효하고 철자가 올바른지 확인합니다.
