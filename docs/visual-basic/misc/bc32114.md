 <span data-ttu-id="1c46a-109">`t2` 에 전달된 형식은 값 형식이어야 하므로 `t1`에 전달된 형식이 `t1` 에 전달된 형식을 구현하거나 상속할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="1c46a-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>
