# &#39;Set&#39; 매개 변수는 &#39;&lt;specifier&gt;&#39;로 선언할 수 없습니다.
`Set` 속성 프로시저에서 매개 변수와 함께 잘못된 한정자를 지정했습니다.  
  
 **오류 ID:** BC31065  
  
### 이 오류를 해결하려면  
  
-   `ByRef`, `ParamArray` 및 `Optional`과 같은 잘못된 한정자를 제거합니다.
