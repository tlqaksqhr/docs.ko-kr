# &#39;&lt;tag&gt;&#39; XML 주석 태그는 &#39;&lt;element&gt;&#39; 언어 요소에서 사용할 수 없습니다.
\<tag\> XML 주석 태그는 '\<element\>' 요소에서 사용할 수 없습니다. XML 주석이 무시됩니다.  
  
 XML 주석 태그가 적용되는 언어 요소에서 지원되지 않습니다.  
  
 **오류 ID:** BC42306  
  
### 이 오류를 해결하려면  
  
-   지원되지 않는 XML 주석 태그를 제거합니다.
