# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;Sub m&#39; 에 두 번 이상 선언 &#39; &lt;네임 스페이스&gt;&#39;: &lt;메시지&gt;
`Sub Main` 은 네임스페이스 내에서 한 번만 선언할 수 있습니다.  
  
 **오류 ID:** BC30738  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   프로젝트에 하나의 `Sub Main` 프로시저만 있어야 합니다.
