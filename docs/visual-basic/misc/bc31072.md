# <a name="warning-treated-as-error-lterrorgt"></a>경고가 오류로 처리: &lt;오류&gt;
**경고를 오류로 처리** 옵션을 설정한 상태에서 컴파일된 어셈블리를 실행할 때 경고가 발생했습니다.  
  
 **오류 ID:** BC31072  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  이 경고를 수정합니다.  
  
2.  **경고를 오류로 처리** 옵션을 해제한 상태에서 어셈블리를 컴파일합니다.
