# &#39;End Interface&#39;는 짝이 되는 &#39;Interface&#39; 뒤에 와야 합니다.
`End Interface` 문이 해당 `Interface` 문 없이 발생합니다.`End Interface`는 해당 `Interface` 문 뒤에 와야 합니다.  
  
 **오류 ID:** BC30252  
  
### 이 오류를 해결하려면  
  
1.  앞의 `Interface` 문이 철자가 잘못되거나 유효하지 않은지 확인합니다.  
  
2.  `Interface` 멤버의 형식이 올바르게 지정되었는지 확인합니다.
