# <a name="option-strict-on-disallows-late-binding"></a>Option Strict On에서는 런타임에 바인딩을 사용할 수 없습니다.
[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] 에서는 모든 데이터 형식을 다른 데이터 형식으로 암시적으로 변환할 수 있습니다. 그러나 한 데이터 형식의 값을 정밀도나 용량이 더 적은 데이터 형식으로 변환할 경우 데이터가 손실될 수 있습니다. `Option Strict On` 을 사용하면 이러한 유형의 변환을 방지할 수 있도록 컴파일 시간 알림이 제공됩니다. 런타임에 바인딩과 함께 `Option Strict On` 을 사용할 수 없습니다.  

 **오류 ID:** BC30574  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   명시적 형식을 사용 하도록 개체 선언을 수정 합니다.  
  
 \- 또는 -  
  
-   명시적 형식을 지정 하려면 런타임에 바인딩 식을 수정 합니다.  
  
 \- 또는 -  
  
-   특정 형식을 유추 하도록 컴파일러를 사용 합니다.  
  
 \- 또는 -  
  
-   뒤에 있는 `Option Strict` 단어를 제거하거나 명시적으로 `On` 를 지정하여 `Off`를 끕니다.
