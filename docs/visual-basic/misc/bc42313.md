# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>XML 주석 태그 &#39;반환&#39; 에서 허용 되지 않습니다는 &#39;WriteOnly&#39; 속성
XML 주석 태그 'returns'는 'WriteOnly' 속성에 사용할 수 없습니다. XML 주석이 무시됩니다.  
  
 쓰기 전용 속성 선언에 대 한 XML 주석에 포함할 수 없습니다는 \<반환 > 태그입니다.  
  
 **오류 ID:** BC42313  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지원 되지 않는 제거 \<반환 > 태그입니다.
