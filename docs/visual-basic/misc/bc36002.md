# <a name="39global39-not-allowed-in-handles-local-name-expected"></a>&#39; 전역 &#39; 핸들;에 사용할 수 없습니다. 로컬 이름이 필요 합니다.
`Handles` 절은 로컬 이벤트를 참조해야 합니다. `Global` 키워드는 전역 프로그래밍 요소에 대한 액세스를 제공합니다.  
  
 **오류 ID:** BC36002  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   전역 인스턴스 대신 이벤트의 로컬 인스턴스를 참조하도록 `Handles` 절을 변경합니다.
