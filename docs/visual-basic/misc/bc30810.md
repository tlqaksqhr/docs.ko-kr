# <a name="methods-cannot-be-declared-39static39"></a>메서드를 선언할 수 없습니다 &#39; 정적 &#39;
`Static` 한정자는 전체 프로시저에 적용할 수 없습니다.  
  
 **오류 ID:** BC30810  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Static` 한정자를 사용하여 프로시저 내의 개별 항목을 static으로 선언합니다.
