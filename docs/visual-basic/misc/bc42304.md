# XML 문서 구문 분석 오류: &lt;error&gt;
이 오류와 함께 다음과 같은 메시지가 나타날 수 있습니다. 자세한 정보는 개별 항목을 참조하세요.  
  
-   [특성이 같은 XML 주석 태그 '\<tag\>'가 동일한 XML 주석 블록에서 두 번 이상 나타납니다.](../../visual-basic/misc/bc42305.md)  
  
-   ['\<tag\>' XML 주석 태그는 '\<element\>' 언어 요소에서 사용할 수 없습니다.](../../visual-basic/misc/bc42306.md)  
  
-   [XML 주석 매개 변수 '\<parameter\>'가 해당 \<keyword\> 문의 매개 변수와 일치하지 않습니다.](../../visual-basic/misc/bc42307.md)  
  
-   [XML 주석 매개 변수에는 'name' 특성이 있어야 함](../../visual-basic/misc/bc42308.md)  
  
-   [XML 주석에 확인할 수 없는 'cref' 특성 '\<attribute\>'가 포함된 태그가 있습니다.](../../visual-basic/misc/bc42309.md)  
  
-   [XML 주석 태그 'include'에는 '\<attribute\>' 특성이 있어야 합니다.](../../visual-basic/misc/bc42310.md)  
  
 **오류 ID:** BC42304
