# <a name="39catch39-block-never-reached-because-39ltname1gt39-inherits-from-39ltname2gt39"></a>&#39;Catch&#39; 블록에 도달 하지 않습니다, 때문에 &#39; &lt;name1&gt; &#39; 에서 상속 &#39; &lt;name2&gt;&#39;
코드 내의 `Catch` 블록에 도달하지 않습니다.  
  
 이 메시지는 기본적으로 경고입니다. 경고를 숨기 거 나 경고를 오류로 처리 하는 방법에 대 한 자세한 내용은 참조 하십시오. [Visual Basic에서 경고 구성](/visualstudio/ide/configuring-warnings-in-visual-basic)합니다.  
  
 **오류 ID:** BC42029  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   불필요한 `Catch` 블록을 제거합니다.
