# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>방법 &#39; &lt;methodname&gt;&#39; 이벤트 &#39; 처리할 수 없는&lt; eventname&gt;&#39; 동일한 서명이 없기 때문에
지정된 이벤트 처리기의 매개 변수 또는 매개 변수 형식이 지정된 이벤트에 대해 정의된 매개 변수 또는 매개 변수 형식과 일치하지 않습니다.  
  
 **오류 ID:** BC31029  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면
