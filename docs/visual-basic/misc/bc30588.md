# 생략된 인수는 ParamArray 매개 변수와 일치할 수 없습니다.
`ParamArray` 매개 변수와 일치하는 데 필요한 인수를 생략했습니다. 프로시저에 임의 개수의 인수를 전달할 수 있도록 하는 `ParamArray`은 인수 목록의 마지막 인수에만 사용할 수 있습니다.  
  
 **오류 ID:** BC30588  
  
### 이 오류를 해결하려면  
  
-   생략된 인수를 제공합니다.
