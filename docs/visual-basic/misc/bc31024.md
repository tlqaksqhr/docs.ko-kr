# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>.NET 런타임 인터페이스를 만들 수 없습니다: &lt;오류&gt;
인터페이스를 만드는 동안 내부 컴파일러 오류가 발생했거나 인터페이스의 메서드 호출이 실패했습니다.  
  
 **오류 ID:** BC31024  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  작업 내용을 저장 하 고 Visual Studio를 다시 시작 합니다.  
  
2.  오류가 다시 발생 하는 경우 Visual Studio를 다시 설치 합니다.  
  
3.  다시 설치 후에도 오류가 계속 발생하면 Microsoft 기술 지원 서비스에 알립니다.
