# &#39;&lt;typename&gt;&#39;은 &#39;MustOverride&#39;로 선언된 메서드를 포함하므로 &#39;MustInherit&#39;으로 선언해야 합니다.
`MustOverride` 멤버를 포함하는 형식이 `MustInherit`로 선언되어야 합니다.  
  
 **오류 ID:** BC31411  
  
### 이 오류를 해결하려면  
  
-   `MustInherit` 한정자를 형식에 추가합니다.
