# 특성 값이 잘못되었습니다. &#39;value&#39;가 필요합니다.
XML 문서 리터럴의 특성 값을 잘못 지정했습니다. XML 문서 리터럴에 대한 유효한 특성 값은 XML 1.0 사양에서 지정됩니다.  
  
 **오류 ID:** BC31181  
  
### 이 오류를 해결하려면  
  
-   XML 1.0 사양에 지정된 유효한 값으로 특성 값을 업데이트합니다.
