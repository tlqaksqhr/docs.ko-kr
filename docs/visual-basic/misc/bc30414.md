# 형식이 다른 두 배열의 차수가 달라 &#39;&lt;type1&gt;&#39; 형식의 값을 &#39;&lt;type2&gt;&#39;로 변환할 수 없습니다.
한 배열을 차원 수가 다른 배열로 변환하려고 했습니다.  
  
 **오류 ID:** BC30414  
  
### 이 오류를 해결하려면  
  
-   차원 수가 일치하도록 하나 또는 두 배열을 변경합니다.
