# <a name="the-source-folder-does-not-exist"></a>소스 폴더가 없습니다.
존재하지 않는 소스 폴더를 참조하는 작업을 시도했습니다. 파일 경로를 잘못 제공했기 때문일 수 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   특히 상대 경로를 사용하는 경우 올바른 경로를 사용하고 있는지 확인합니다.
