# <a name="indirect-constraint-39ltconstraint1gt39-obtained-from-the-type-parameter-constraint-39lttypeparameter1gt39-conflicts-with-the-constraint-39ltconstraint2gt39"></a>간접 제약 조건 &#39; &lt;t 1&gt; &#39; 형식 매개 변수 제약 조건에서 가져온 &#39; &lt;typeparameter1&gt; &#39; 제약 조건과 충돌 &#39; &lt;constraint2&gt;&#39;
제네릭 형식이 직접 및 간접 제약 조건의 조합으로 인해 충돌하는 제약 조건으로 선언되었습니다.  
  
 다음 문은 이 오류를 생성할 수 있습니다.  
  
 `Public Class testClass(Of t1 As {t2, Class}, t2 As Structure)`  
  
 간접 제약 조건 `Structure` 및 직접 제약 조건 `Class` 으로 인해 형식 매개 변수 `t1`에 대한 충돌이 발생합니다. 그 이유는 해당 형식 인수가 `Structure` 제약 조건에서는 값 형식이어야 하고 `Class` 에서는 참조 형식이어야 하기 때문입니다.  
  
 **오류 ID:** BC32111  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제약 조건의 충돌을 방지하려면 형식 매개 변수 제약 조건을 변경합니다.
