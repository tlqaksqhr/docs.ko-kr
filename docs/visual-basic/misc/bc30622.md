# &#39;End Module&#39;은 짝이 되는 &#39;Module&#39; 뒤에 와야 합니다.
모듈 블록이 `Module` 문으로 시작하여 `End Module` 구문으로 끝납니다.  
  
 **오류 ID:** BC30622  
  
### 이 오류를 해결하려면  
  
-   모듈 블록이 `Module` 문으로 시작하는지 확인합니다.
