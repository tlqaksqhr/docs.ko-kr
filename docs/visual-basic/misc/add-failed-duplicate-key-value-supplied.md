# <a name="add-failed-duplicate-key-value-supplied"></a>추가하지 못했습니다. 중복 키 값이 제공되었습니다.
키 값이 다른 키 값과 같기 때문에 `Add` 작업에 실패했습니다. 키 값은 고유해야 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   키 값이 고유한지 확인합니다.
