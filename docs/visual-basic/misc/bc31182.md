# <a name="attribute-value-is-not-valid-expecting-39value139-or-39value239"></a>특성 값이 잘못 되었습니다. 예상 &#39; value1 &#39; 또는 &#39; value2 &#39;
XML 문서 리터럴의 특성 값을 잘못 지정했습니다. XML 문서 리터럴에 대한 올바른 특성 값은 XML 1.0 사양을 기반으로 합니다.  
  
 **오류 ID:** BC31182  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 1.0 사양에 따라 유효한 값으로 특성 값을 변경합니다.
