# <a name="constructor-must-be-declared-as-a-sub-not-as-a-function"></a>생성자는 Function이 아니라 Sub로 선언되어야 합니다.
`Function New`를 선언하려고 했습니다. 생성자는 `Sub New`로 선언되어야 합니다.  
  
 **오류 ID:** BC30493  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Sub` 대신 `Function`를 사용합니다.
