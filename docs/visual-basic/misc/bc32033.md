# <a name="39end-get39-must-be-the-first-statement-on-a-line"></a>&#39;End g e&#39; 줄에서 첫 번째 문 이어야 합니다
`End Get` 문 앞에 콜론(:) 문 구분 기호가 있습니다. `End Get` 은 해당 소스 줄에서 유일한 문이어야 합니다.  
  
 **오류 ID:** BC32033  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   각 문을 서로 다른 줄로 분리합니다.
