# &#39;RaiseEvent&#39; 선언은 짝이 되는 &#39;End RaiseEvent&#39;로 끝나야 합니다.
`RaiseEvent` 선언은 `End RaiseEvent` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31117  
  
### 이 오류를 해결하려면  
  
-   `RaiseEvent` 선언이 `End RaiseEvent` 문으로 끝나는지 확인합니다.
