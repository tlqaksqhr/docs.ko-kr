# &#39;RaiseEvent&#39;가 이미 선언되었습니다.
둘 이상의 `RaiseEvent` 선언이 사용자 지정 이벤트 선언에 표시되었습니다.`RaiseEvent` 선언은 이벤트를 발생시키는 데 사용되는 프로시저를 선언합니다.  
  
 **오류 ID:** BC31129  
  
### 이 오류를 해결하려면  
  
-   중복 `RaiseEvent` 문을 제거합니다.
