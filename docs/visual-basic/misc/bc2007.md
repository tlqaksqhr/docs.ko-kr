# <a name="unrecognized-option-ltoptionnamegt-ignored"></a>인식할 수 없는 옵션 &lt;a m e&gt;의 경우 무시 됩니다.
명령줄에서 잘못된 옵션을 지정했습니다.  
  
 **오류 ID:** BC2007  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지정된 옵션을 확인하여 잘못된 옵션을 찾습니다.
