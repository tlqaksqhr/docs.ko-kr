# <a name="expected-closing-39--gt39-for-xml-comment"></a>닫는 예상 &#39;-&gt; &#39; XML 주석용
XML 주석 리터럴에 필요한 닫는 '-->' 문자가 없습니다.  
  
 **오류 ID:** BC31161  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 주석 리터럴의 끝에 필요한 닫는 '-->' 문자를 추가합니다.
