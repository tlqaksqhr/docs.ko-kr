# <a name="39on-goto39-and-39on-gosub39-statements-are-no-longer-supported"></a>&#39; on GoTo &#39; 및 &#39; On GoSub &#39; 문은 더 이상 지원
더 이상 변수 또는 식의 값을 `GoTo` 및 `GoSub` 문과 함께 사용하여 프로그램 흐름을 제어할 수 없습니다.  
  
 **오류 ID:** BC30817  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `If...Then...Else` 또는 `Case` 문을 사용하도록 응용 프로그램을 재구성합니다.
