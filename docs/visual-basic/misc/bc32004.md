# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39; &lt;이름&gt;&#39; 프로젝트 &#39;에서 선언 된&lt; projectname1&gt;&#39; 프로젝트 &#39; 참조 되지 않은&lt; projectname2&gt;&#39;
현재 프로젝트에 참조가 없는 프로그래밍 요소가 프로젝트에 정의되어 있습니다.  
  
 **오류 ID:** BC32004  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   현재 프로젝트에서 원하는 요소를 포함하는 프로젝트에 대한 참조를 추가합니다.
