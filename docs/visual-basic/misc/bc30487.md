# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>연산자 &#39; &lt;a m e&gt;&#39; 형식의 &#39;에 대해 정의 되지 않은&lt; typename&gt;&#39;

 **오류 ID:** BC30487  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   단항 연산자를 식에 적절한 연산자로 바꿉니다.
