# 변수와 그 형식 모두에 null 허용 한정자를 지정할 수 없습니다.
nullable 형식 한정자\(?\)가 선언된 변수와 선언된 형식에 모두 포함되어 있습니다.  
  
 **오류 ID:** BC33100  
  
### 이 오류를 해결하려면  
  
1.  변수 또는 선언된 형식에서 nullable 형식 한정자\(?\)를 제거합니다.
