# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>'&lt;procedurename&gt;'에 지원 되지 않는 반환 형식 또는 지원 되지 않는 매개 변수 형식
매개 변수 또는 반환 형식으로 표현할 수 없는 사용 하 여 프로시저 선언 [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]합니다.  
  
 **오류 ID:** BC30657  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지원되는 형식으로 반환 형식 또는 매개 변수를 정의합니다.  
  
-   이 프로시저에서 사용되는 데이터를 설명하는 새 형식을 개발합니다.
