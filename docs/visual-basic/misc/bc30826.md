# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39; EndIf &#39; 문은 더 이상 지원. 사용 &#39; End If &#39; 대신
`EndIf` 문이 더 이상 지원되지 않습니다.  
  
 **오류 ID:** BC30826  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `End If` 를 사용하여 `If` 문을 종료합니다.
