# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>문자 시퀀스 &#39;-&#39; XML 주석에서 허용 되지 않습니다.
문자 시퀀스 '--'를 포함하는 XML 주석 리터럴은 지원되지 않습니다.  
  
 **오류 ID:** BC31171  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   XML 주석 리터럴에서 문자 시퀀스 '--'를 제거합니다.
