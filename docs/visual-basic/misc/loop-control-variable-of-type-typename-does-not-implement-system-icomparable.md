# <a name="loop-control-variable-of-type-39lttypenamegt39-does-not-implement-the-39systemicomparable39-interface"></a>루프 제어 변수 형식의 &#39; &lt;typename&gt;&#39; 구현 하지 않습니다는 &#39; System.IComparable &#39; 인터페이스
변수는 `System.IComparable` 인터페이스를 구현하지 않는 형식입니다. `System.IComparable` 은 값 형식이나 클래스에서 구현되어야 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  변수가 값 형식인지 확인합니다.
