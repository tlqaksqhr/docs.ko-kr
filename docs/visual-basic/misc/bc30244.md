# <a name="39ltmodifiergt39-is-not-valid-on-a-declare"></a>&#39; &lt;한정자&gt;&#39; Declare에서 유효 하지 않습니다
`Declare` 문에 `ReadOnly`같은 잘못된 키워드가 포함되어 있습니다.  
  
 **오류 ID:** BC30244  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Declare` 문에서 잘못된 키워드를 제거합니다.
