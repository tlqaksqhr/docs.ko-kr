# &#39;&lt;functionname&gt;&#39;이 선언되지 않았습니다(Visual Basic 오류).
'\<functionname\>'이 선언되지 않았습니다. 파일 I\/O 기능은 'Microsoft.VisualBasic' 네임스페이스에서 사용할 수 있습니다.  
  
 **오류 ID:** BC30815  
  
### 이 오류를 해결하려면  
  
-   `My.Computer.FileSystem` `System.IO` 네임스페이스, Visual Basic 런타임 함수 또는 `Microsoft.VisualBasic.FileSystem` 개체에 정의된 파일 I\/O 함수를 사용합니다.
