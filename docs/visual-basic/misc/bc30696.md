# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>&#39; &lt;membername1&gt;&#39; 및 &#39;&lt; m e 2&gt;&#39; 선택적 매개 변수의 형식만 다르므로 서로 오버 로드할 수 없습니다
오버로드된 멤버는 하나 이상의 필수 매개 변수에 서로 다른 데이터 형식이 있어야 합니다.  
  
 **오류 ID:** BC30696  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   필수 매개 변수 중 하나 이상을 다른 데이터 형식으로 정의합니다.
