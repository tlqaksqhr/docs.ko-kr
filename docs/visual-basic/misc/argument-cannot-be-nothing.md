# <a name="argument-cannot-be-nothing"></a>인수는 Nothing일 수 없습니다.
값이 있어야 하는 인수에 대해 null 값이 제공되었습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   개체의 인스턴스를 제공하지 않고 개체를 사용하려고 했을 수 있습니다. 이런 경우 `New` 키워드를 사용하여 인스턴스를 만듭니다.  
  
-   값이 올바르게 계산되었는지 확인합니다.
