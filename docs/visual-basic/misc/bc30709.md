# 직접 실행 창에서는 변수 선언 문을 사용할 수 없습니다.
**직접 실행** 창에서는 변수 선언을 사용할 수 없습니다.  
  
 **오류 ID:** BC30709  
  
### 이 오류를 해결하려면  
  
-   **직접 실행** 창에서 변수를 선언하지 마세요.
