# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>WithEvents 변수 이름 길이는 1019자를 초과할 수 없습니다.
`WithEvents` 을 사용하여 선언된 모든 변수의 이름 또는 식별자는 1019자로 제한됩니다.  
  
 **오류 ID:** BC32205  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   식별자의 길이를 줄입니다.
