# &lt;optionname&gt; 옵션에 &lt;argument&gt;가 필요합니다.
필수 인수가 생략되었습니다.  
  
 **오류 ID:** BC2006  
  
### 이 오류를 해결하려면  
  
-   누락된 인수를 제공합니다.
