# <a name="39addressof39-expressions-are-not-valid-in-the-first-expression-of-a-39select-case39-statement"></a>&#39;AddressOf&#39; 식의 첫 번째 식에 유효 하지 않은 한 &#39;Select Case&#39; 문
`AddressOf` 문에서 테스트 식에 대해 `Select Case` 식을 사용할 수 없습니다. `AddressOf` 식이 함수 대리자를 반환하고 `Select Case` 문의 테스트 식이 기본 데이터 형식이어야 합니다.  
  
 **오류 ID:** BC36636  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   코드를 검사하여 `If...Then...Else` 문과 같은 다른 조건부 생성이 적합한지 결정합니다.
