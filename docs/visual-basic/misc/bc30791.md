`Declare` 선언에 `Structure` 선언에서 유효하지 않은 지정자가 있습니다.  
  
 **오류 ID:** BC30791  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   지정자를 제거합니다.
