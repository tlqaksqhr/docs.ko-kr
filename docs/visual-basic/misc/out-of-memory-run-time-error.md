# 메모리가 부족합니다(Visual Basic 런타임 오류).
사용 가능한 것보다 더 많은 메모리가 필요했습니다.  
  
### 이 오류를 해결하려면  
  
1.  열려 있는 불필요한 응용 프로그램, 문서 또는 소스 파일을 닫습니다.  
  
2.  큰 모듈 또는 프로시저를 더 작은 모듈 또는 프로시저로 나눕니다.  
  
3.  TSR\(Terminate\-and\-Stay\-Resident\) 프로그램을 제거합니다.  
  
4.  불필요한 장치 드라이버를 제거합니다.  
  
5.  공용 변수 수를 줄입니다.
