# <a name="lttype1gt-39lttypenamegt39-and-lttype2gt-39lttypenamegt39-conflict-in-ltnamespacegt-39ltnamespacenamegt39"></a>&lt;type1&gt; &#39;&lt; typename&gt;&#39; 및 &lt;type2&gt; &#39;&lt; typename&gt;&#39;에서 충돌 &lt;네임 스페이스&gt; &#39;&lt; namespacename&gt;&#39;
클래스 및 모듈 등 선언된 두 요소에 동일한 네임스페이스의 동일한 이름이 있습니다.  
  
 **오류 ID:** BC30179  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   한 요소의 선언을 다른 네임스페이스로 이동합니다.  
  
-   한 요소의 이름을 변경합니다.
