# &#39;Public&#39;, &#39;Private&#39;, &#39;Protected&#39;, &#39;Friend&#39; 또는 &#39;Protected Friend&#39; 중 하나만 지정할 수 있습니다.
요소 선언에서 충돌하는 접근성이 발생합니다. 접근성은 하나만 지정할 수 있습니다.  
  
 **오류 ID:** BC30176  
  
### 이 오류를 해결하려면  
  
-   하나의 접근성을 선택하고 선언에서 다른 키워드를 제거합니다.
