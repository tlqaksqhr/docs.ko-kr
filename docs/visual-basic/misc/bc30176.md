# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>중 하나에만 &#39;공용&#39;, &#39;개인&#39;, &#39;Protected&#39;, &#39;Friend&#39;, 또는 &#39;Protected Friend&#39; 지정할 수 있습니다
요소 선언에서 충돌하는 접근성이 발생합니다. 접근성은 하나만 지정할 수 있습니다.  
  
 **오류 ID:** BC30176  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   하나의 접근성을 선택하고 선언에서 다른 키워드를 제거합니다.
