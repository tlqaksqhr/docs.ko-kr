# <a name="implicit-conversion-from-39lttypename1gt39-to-39lttypename2gt39"></a>암시적으로 변환 &#39; &lt;typename1&gt; &#39; 를 &#39; &lt;typename2&gt;&#39;
식 또는 대입문이 한 데이터 형식의 값을 받아서 다른 형식으로 변환합니다. 변환 키워드를 사용하지 않으므로 변환은 *암시적*입니다.  
  
 이 메시지는 기본적으로 경고입니다. 경고를 숨기거나 오류로 처리하는 방법에 대한 자세한 내용은 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)을 참조하세요.  
  
 **오류 ID:** BC42016  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   가능한 경우 모든 변환을 수행 하지 않아도 Visual Basic 같은 데이터 형식의 값 사용.  
  
-   변환이 `CType` 명시적 *이 되도록*또는 다른 변환 키워드 중 하나를 사용합니다.
