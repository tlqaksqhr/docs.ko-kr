# &#39;&lt;argumentname&gt;&#39; 인수가 &#39;&lt;typename&gt;&#39; 형식으로 변환할 수 없습니다.
변환이 인수를 잘못된 형식으로 변경하려고 합니다.  
  
### 이 오류를 해결하려면  
  
-   올바른 형식으로 변경합니다.
