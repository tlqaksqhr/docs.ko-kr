# <a name="39nothing39-cannot-be-evaluated"></a>&#39; 아무것도 &#39; 평가할 수 없습니다.
디버그 창에서 `Nothing` 키워드가 사용되었습니다.  
  
 **오류 ID:** BC30701  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   디버그하는 동안 `Nothing` 을 사용하지 마세요.
