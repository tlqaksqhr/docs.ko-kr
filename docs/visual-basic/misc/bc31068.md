# <a name="delegate-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>인터페이스의 대리자를 선언할 수 없습니다 &#39; &lt;지정자&gt;&#39;
잘못된 한정자가 인터페이스에서 선언된 대리자로 지정되었습니다.  
  
 **오류 ID:** BC31068  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`및 `Overridable`과 같은 한정자를 제거합니다.
