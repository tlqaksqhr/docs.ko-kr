# <a name="constructors-cannot-implement-interface-methods"></a>생성자는 인터페이스 메서드를 구현할 수 없습니다.
`New` 키워드를 사용하여 잘못된 인터페이스 메서드를 구현하려고 했습니다.  
  
 **오류 ID:** BC30550  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Implements` 키워드를 사용하여 인터페이스를 구현합니다.
