# &#39;AddHandler&#39; 및 &#39;RemoveHandler&#39; 메서드 매개 변수는 &#39;ByRef&#39;로 선언할 수 없습니다.
`AddHandler` 및 `RemoveHandler` 메서드의 매개 변수는 `ByRef` 한정자로 선언할 수 없습니다.  
  
 **오류 ID:** BC31134  
  
### 이 오류를 해결하려면  
  
-   매개 변수에서 `ByRef` 키워드를 제거합니다.
