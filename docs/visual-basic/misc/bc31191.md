# Win32 매니페스트를 포함하는 동안 오류가 발생했습니다. &lt;manifest&gt;
`/win32manifest` 컴파일러 옵션에 대해 지정된 파일은 포함될 수 없습니다. 이는 매니페스트 파일에 대해 보안 권한이 충분하지 않기 때문일 수 있습니다.  
  
 **오류 ID:** BC31191  
  
### 이 오류를 해결하려면  
  
1.  Visual Basic 컴파일러 명령을 실행하는 ID는 Win32 매니페스트 파일에 대한 액세스 권한이 있어야 합니다.
