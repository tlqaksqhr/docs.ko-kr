# <a name="accessing-application-forms-visual-basic"></a>응용 프로그램 폼 액세스(Visual Basic)
`My.Forms` 개체는 응용 프로그램 프로젝트에 선언된 각 Windows Form의 인스턴스에 액세스하는 간편한 방법을 제공합니다. `My.Application` 개체의 속성을 사용하여 응용 프로그램의 시작 화면 및 기본 폼에 액세스하고 응용 프로그램의 열려 있는 폼 목록을 가져올 수도 있습니다.  
  
## <a name="tasks"></a>작업  
 다음 표에는 응용 프로그램의 양식에 액세스하는 방법을 보여 주는 예제가 나와 있습니다.  
  
|후|참조|  
|---|---|  
|응용 프로그램의 다른 폼에서 한 폼에 액세스합니다.|[My.Forms 개체](../../../visual-basic/language-reference/objects/my-forms-object.md)|  
|응용 프로그램의 열려 있는 모든 폼의 제목을 표시합니다.|<xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase.OpenForms%2A>|  
|응용 프로그램이 시작될 때 시작 화면을 상태 정보로 업데이트합니다.|<xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase.SplashScreen%2A>|
