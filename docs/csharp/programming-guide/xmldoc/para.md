# &lt;para&gt;(C# 프로그래밍 가이드)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

## 구문  
  
```  
<para>content</para>  
```  
  
#### 매개 변수  
 `content`  
 단락 텍스트입니다.  
  
## 설명  
 \<para\> 태그를 [\<summary\>](../Topic/%3Csummary%3E%20\(C%23%20Programming%20Guide\).md), [\<remarks\>](../../../csharp/programming-guide/xmldoc/remarks.md) 또는 [\<returns\>](../../../csharp/programming-guide/xmldoc/returns.md) 같은 태그 내에서 사용하여 텍스트에 구문을 추가할 수 있습니다.  
  
 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md)로 컴파일하여 문서 주석을 파일로 저장합니다.  
  
## 예제  
 \<para\> 사용 예제는 [\<summary\>](../Topic/%3Csummary%3E%20\(C%23%20Programming%20Guide\).md)를 참조하십시오.
