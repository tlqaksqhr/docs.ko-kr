# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="745a1-102">컴파일러 경고(수준 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="745a1-102">Compiler Warning (level 2) CS0108</span></span>
<span data-ttu-id="745a1-103">‘member1’은(는) 상속된 ‘member2’ 멤버를 숨깁니다.</span><span class="sxs-lookup"><span data-stu-id="745a1-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="745a1-104">숨기려면 new 키워드를 사용하세요.</span><span class="sxs-lookup"><span data-stu-id="745a1-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="745a1-105">기본 클래스의 변수와 동일한 이름을 사용하여 변수가 선언되었습니다.</span><span class="sxs-lookup"><span data-stu-id="745a1-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="745a1-106">그러나 [new](../../../csharp/language-reference/keywords/new.md) 키워드가 사용되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="745a1-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="745a1-107">이 경고는 **new**를 사용해야 한다고 알립니다. 선언에 **new**를 사용한 것처럼 변수가 선언되었습니다.</span><span class="sxs-lookup"><span data-stu-id="745a1-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="745a1-108">다음 샘플에서는 CS0108을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="745a1-108">The following sample generates CS0108:</span></span>  
  
```csharp  
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
