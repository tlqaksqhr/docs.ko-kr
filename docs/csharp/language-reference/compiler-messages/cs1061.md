# <a name="compiler-error-cs1061"></a><span data-ttu-id="439ba-102">컴파일러 오류 CS1061</span><span class="sxs-lookup"><span data-stu-id="439ba-102">Compiler Error CS1061</span></span>
<span data-ttu-id="439ba-103">'type'에 'member'에 대한 정의가 없고 'type' 형식의 첫 번째 인수를 허용하는 확장 메서드 'name'을(를) 찾을 수 없습니다. using 지시문 또는 어셈블리 참조가 있는지 확인하세요.</span><span class="sxs-lookup"><span data-stu-id="439ba-103">'type' does not contain a definition for 'member' and no extension method 'name' accepting a first argument of type 'type' could be found (are you missing a using directive or an assembly reference?).</span></span>  
  
 <span data-ttu-id="439ba-104">메서드를 호출하거나 존재하지 않는 클래스 멤버에 액세스하려고 하면 이 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="439ba-104">This error occurs when you try to call a method or access a class member that does not exist.</span></span>  
  
## <a name="example"></a><span data-ttu-id="439ba-105">예</span><span class="sxs-lookup"><span data-stu-id="439ba-105">Example</span></span>  
 <span data-ttu-id="439ba-106">다음 예제에서는 `TestClass1`에 `DisplaySomething` 메서드가 없기 때문에 CS1061을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="439ba-106">The following example generates CS1061 because `TestClass1` does not have a `DisplaySomething` method.</span></span> <span data-ttu-id="439ba-107">`WriteSomething`이라는 메서드가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="439ba-107">It does have a method that is called `WriteSomething`.</span></span> <span data-ttu-id="439ba-108">이 소스 코드의 작성자가 쓰려고 한 내용입니다.</span><span class="sxs-lookup"><span data-stu-id="439ba-108">Perhaps that is what the author of this source code meant to write.</span></span>  
  
```csharp  
// cs1061.cs  
public class TestClass1  
{  
    // TestClass1 has one method, called WriteSomething.  
    public void WriteSomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestClass2  
{  
    // TestClass2 has one method, called DisplaySomething.  
    public void DisplaySomething(string s)  
    {  
        System.Console.WriteLine(s);  
    }  
}  
  
public class TestTheClasses  
{  
    public static void Main()  
    {  
        TestClass1 tc1 = new TestClass1();  
        TestClass2 tc2 = new TestClass2();  
        // The following call fails because TestClass1 does not have   
        // a method called DisplaySomething.  
        tc1.DisplaySomething("Hello");      // CS1061  
  
        // To correct the error, change the method call to either   
        // tc1.WriteSomething or tc2.DisplaySomething.  
        tc1.WriteSomething("Hello from TestClass1");  
        tc2.DisplaySomething("Hello from TestClass2");  
    }  
}  
```
