# <a name="compiler-error-cs1501"></a><span data-ttu-id="5885a-102">컴파일러 오류 CS1501</span><span class="sxs-lookup"><span data-stu-id="5885a-102">Compiler Error CS1501</span></span>
<span data-ttu-id="5885a-103">'number'개의 인수를 사용하는 'method' 메서드에 대한 오버로드가 없습니다.</span><span class="sxs-lookup"><span data-stu-id="5885a-103">No overload for method 'method' takes 'number' arguments</span></span>  
  
 <span data-ttu-id="5885a-104">클래스 메서드를 호출했지만 메서드의 정의가 지정된 수의 인수를 사용하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="5885a-104">A call was made to a class method, but no definition of the method takes the specified number of arguments.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5885a-105">예제</span><span class="sxs-lookup"><span data-stu-id="5885a-105">Example</span></span>  
 <span data-ttu-id="5885a-106">다음 샘플에서는 CS1501을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5885a-106">The following sample generates CS1501.</span></span>  
  
```csharp  
using System;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            ExampleClass ec = new ExampleClass();  
            ec.ExampleMethod();  
            ec.ExampleMethod(10);  
            // The following line causes compiler error CS1501 because   
            // ExampleClass does not contain an ExampleMethod that takes  
            // two arguments.  
            ec.ExampleMethod(10, 20);  
        }  
    }  
  
    // ExampleClass contains two overloads for ExampleMethod. One of them   
    // has no parameters and one has a single parameter.  
    class ExampleClass  
    {  
        public void ExampleMethod()  
        {  
            Console.WriteLine("Zero parameters");  
        }  
  
        public void ExampleMethod(int i)  
        {  
            Console.WriteLine("One integer parameter.");  
        }  
  
        //// To fix the error, you must add a method that takes two arguments.  
        //public void ExampleMethod (int i, int j)  
        //{  
        //    Console.WriteLine("Two integer parameters.");  
        //}  
    }  
}  
```
