# <a name="compiler-error-cs0504"></a><span data-ttu-id="089ee-102">컴파일러 오류 CS0504</span><span class="sxs-lookup"><span data-stu-id="089ee-102">Compiler Error CS0504</span></span>
<span data-ttu-id="089ee-103">‘variable’ 상수는 static으로 표시할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="089ee-103">The constant 'variable' cannot be marked static</span></span>  
  
 <span data-ttu-id="089ee-104">변수가 [const](../../../csharp/language-reference/keywords/const.md)이면 [static](../../../csharp/language-reference/keywords/static.md)이기도 합니다.</span><span class="sxs-lookup"><span data-stu-id="089ee-104">If a variable is [const](../../../csharp/language-reference/keywords/const.md), it is also [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="089ee-105">**const** 및 **static** 변수를 원하는 경우 해당 변수를 **const**로 선언하면 됩니다. **static** 변수만 원하는 경우 **static**으로 표시합니다.</span><span class="sxs-lookup"><span data-stu-id="089ee-105">If you want a **const** and **static** variable, just declare that variable as **const**; if all you want is a **static** variable, just mark it **static**.</span></span>  
  
 <span data-ttu-id="089ee-106">다음 샘플에서는 CS0504를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="089ee-106">The following sample generates CS0504:</span></span>  
  
```csharp  
// CS0504.cs  
namespace x  
{  
   abstract public class clx  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const  
      abstract public void f();  
   }  
}  
```
