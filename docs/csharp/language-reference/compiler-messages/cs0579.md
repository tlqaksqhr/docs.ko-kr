# <a name="compiler-error-cs0579"></a><span data-ttu-id="761d0-102">컴파일러 오류 CS0579</span><span class="sxs-lookup"><span data-stu-id="761d0-102">Compiler Error CS0579</span></span>
<span data-ttu-id="761d0-103">'attribute' 특성이 중복되었습니다.</span><span class="sxs-lookup"><span data-stu-id="761d0-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="761d0-104">특성의 [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md)에 **AllowMultiple=true**가 지정되지 않은 경우 동일한 특성을 두 번 이상 지정할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="761d0-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="761d0-105">예</span><span class="sxs-lookup"><span data-stu-id="761d0-105">Example</span></span>  
 <span data-ttu-id="761d0-106">다음 예제에서는 CS0579를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="761d0-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
