# Compiler Error CS0201
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

할당, 호출, 증가, 감소 및 새 개체 식만 문으로 사용할 수 있습니다.  
  
 컴파일러에서 잘못된 문을 발견하여 오류를 생성합니다.  잘못된 문은 할당\([\=](../../../csharp/language-reference/operators/assignment-operator.md)\), 메서드 호출 [\(\)](../../../csharp/language-reference/operators/invocation-operator.md), [new](../../../csharp/language-reference/keywords/new.md), [\-\-](../../../csharp/language-reference/operators/decrement-operator.md) 또는 [\+\+](../../../csharp/language-reference/operators/increment-operator.md) 연산을 나타내지 않는 세미콜론으로 끝나는 하나 이상의 줄입니다.  자세한 내용은 [문, 식, 연산자](../../../visual-basic/reference/command-line-compiler/index.md)을 참조하십시오.  
  
## 예제  
 다음 샘플에서는 2 \* 3이 문이 아닌 식이기 때문에 C0201 오류가 발생하는 경우를 보여 줍니다.  코드가 컴파일되게 하려면 변수에 식 값을 할당해 보십시오.  
  
```  
// CS0201.cs  
public class MainClass  
{  
   public static void Main()  
   {  
      2 * 3;   // CS0201  
      // Try the following line instead.  
      //   int i = 2 * 3;  
   }  
}  
```  
  
## 예제  
 다음 샘플에서는 증가 연산으로 인해 매개 변수화되지 않더라도 문이 아니기 때문에 CS0201 오류가 발생하는 경우를 보여 줍니다.  
  
```  
// CS0201_b.cs  
// compile with: /target:library  
public class MyList<T>   
{  
   public void Add(T x)  
   {  
      int i = 0;  
      if ( (object)x == null)  
      {  
         checked(i++);   // CS0201  
  
         // OK  
         checked {  
            i++;   
         }  
      }  
   }  
}  
```
