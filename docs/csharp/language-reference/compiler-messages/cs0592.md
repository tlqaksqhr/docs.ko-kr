# <a name="compiler-error-cs0592"></a><span data-ttu-id="efaf7-102">컴파일러 오류 CS0592</span><span class="sxs-lookup"><span data-stu-id="efaf7-102">Compiler Error CS0592</span></span>
<span data-ttu-id="efaf7-103">이 선언 형식에서는 ‘attribute’ 특성이 유효하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="efaf7-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="efaf7-104">이 특성은 ‘type’ 선언에서만 유효합니다.</span><span class="sxs-lookup"><span data-stu-id="efaf7-104">It is valid on 'type' declarations only.</span></span>  
  
 <span data-ttu-id="efaf7-105">특성을 정의할 때 `AttributeTargets` 값을 지정하여 적용할 수 있는 구조체를 정의합니다.</span><span class="sxs-lookup"><span data-stu-id="efaf7-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="efaf7-106">다음 예제에서는 `AttributeUsage` 특성이 `AttributeTargets.Interface`를 지정하기 때문에 `MyAttribute` 특성을 인터페이스에만 적용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="efaf7-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Interface`.</span></span> <span data-ttu-id="efaf7-107">특성이 클래스(`A` 클래스)에 적용되기 때문에 오류가 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="efaf7-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="efaf7-108">예</span><span class="sxs-lookup"><span data-stu-id="efaf7-108">Example</span></span>  
 <span data-ttu-id="efaf7-109">다음 샘플에서는 CS0592를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="efaf7-109">The following sample generates CS0592:</span></span>  
  
```csharp  
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute   
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.   
public class A    
{  
    public static void Main()  
    {  
    }  
}  
```
