## <a name="example"></a><span data-ttu-id="78a80-106">예</span><span class="sxs-lookup"><span data-stu-id="78a80-106">Example</span></span>

## <a name="example"></a><span data-ttu-id="78a80-108">예</span><span class="sxs-lookup"><span data-stu-id="78a80-108">Example</span></span>  
 <span data-ttu-id="78a80-109">다음 샘플에서는 특성을 정의하는 방법을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="78a80-109">The following sample shows how you might define an attribute:</span></span>  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
