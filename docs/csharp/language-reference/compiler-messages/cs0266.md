# <a name="compiler-error-cs0266"></a><span data-ttu-id="cea48-102">컴파일러 오류 CS0266</span><span class="sxs-lookup"><span data-stu-id="cea48-102">Compiler Error CS0266</span></span>
<span data-ttu-id="cea48-103">암시적으로 'type1' 형식을 'type2' 형식으로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="cea48-103">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="cea48-104">명시적 변환이 있습니다. 캐스트가 있는지 확인하세요.</span><span class="sxs-lookup"><span data-stu-id="cea48-104">An explicit conversion exists (are you missing a cast?)</span></span>  
  
 <span data-ttu-id="cea48-105">이 오류는 코드에서 암시적으로 변환할 수 없는 두 형식 간에 변환하려고 하지만 명시적 변환을 사용할 수 있는 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="cea48-105">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="cea48-106">자세한 내용은 [캐스팅 및 형식 변환](../../../csharp/programming-guide/types/casting-and-type-conversions.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="cea48-106">For more information, see [Casting and Type Conversions](../../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="cea48-107">다음 코드는 CS0266을 생성하는 예제입니다.</span><span class="sxs-lookup"><span data-stu-id="cea48-107">The following code shows examples that generate CS0266.</span></span>  
  
```csharp  
// CS0266.cs  
class MyClass  
{  
    public static void Main()  
    {  
        // You cannot implicitly convert a double to an integer.  
        double d = 3.2;  
  
        // The following line causes compiler error CS0266.  
        int i1 = d;  
  
        // However, you can resolve the error by using an explicit conversion.  
        int i2 = (int)d;  
  
        // You cannot implicitly convert an object to a class type.  
        object obj = "MyString";  
  
        // The following assignment statement causes error CS0266.  
        MyClass myClass = obj;    
  
        // You can resolve the error by using an explicit conversion.  
        MyClass c = ( MyClass )obj;  
  
        // You cannot implicitly convert a base class object to a derived class type.  
        MyClass mc = new MyClass();  
        DerivedClass dc = new DerivedClass();  
  
        // The following line causes compiler error CS0266.  
        dc = mc;  
  
        // You can resolve the error by using an explicit conversion.  
        dc = (DerivedClass)mc;  
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
