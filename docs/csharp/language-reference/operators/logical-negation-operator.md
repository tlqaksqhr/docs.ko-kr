# <a name="-operator-c-reference"></a>! 연산자(C# 참조)
논리적 부정 연산자(`!`)는 해당 피연산자를 부정하는 단항 연산자입니다. `bool`에 대해 정의되며, 해당 피연산자가 `false`인 경우에만 `true`를 반환합니다.
