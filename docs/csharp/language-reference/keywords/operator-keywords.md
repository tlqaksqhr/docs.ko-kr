# <a name="operator-keywords-c-reference"></a>연산자 키워드(C# 참조)
개체 생성, 개체의 런타임 형식 확인, 형식의 크기 가져오기 등의 기타 작업을 수행하는 데 사용됩니다. 이 섹션에서는 다음과 같은 키워드를 소개합니다.  
  
-   [as](../../../csharp/language-reference/keywords/as.md) 개체를 호환되는 형식으로 변환합니다.  
  
-   [await](../../../csharp/language-reference/keywords/await.md) 대기 작업이 완료될 때까지 비동기 메서드를 일시 중단합니다.  
  
-   [is](../../../csharp/language-reference/keywords/is.md) 개체의 런타임 형식을 확인합니다.  
  
-   [new](../../../csharp/language-reference/keywords/new.md)  
  
    -   [new 연산자](../../../csharp/language-reference/keywords/new-operator.md) 개체를 만듭니다.  
  
    -   [new 한정자](../../../csharp/language-reference/keywords/new-modifier.md) 상속된 멤버를 숨깁니다.  
  
    -   [new 제약 조건](../../../csharp/language-reference/keywords/new-constraint.md) 형식 매개 변수를 한정합니다.  
  
-   [nameof](nameof.md) 변수, 형식 또는 멤버의 단순(정규화되지 않은) 문자열 이름을 가져옵니다.
 
-   [sizeof](../../../csharp/language-reference/keywords/sizeof.md) 형식의 크기를 가져옵니다.  
  
-   [typeof](../../../csharp/language-reference/keywords/typeof.md) 형식에 대한 **System.Type** 개체를 가져옵니다.  
  
-   [true](../../../csharp/language-reference/keywords/true.md)  
  
    -   [true 연산자](../../../csharp/language-reference/keywords/true-operator.md) true를 나타내려면 부울 값 true를 반환하고, 그렇지 않으면 false를 반환합니다.  
  
    -   [true 리터럴](../../../csharp/language-reference/keywords/true-literal.md) 부울 값 true를 나타냅니다.  
  
-   [false](../../../csharp/language-reference/keywords/false.md)  
  
    -   [false 연산자](../../../csharp/language-reference/keywords/false-operator.md) false를 나타내려면 부울 값 true를 반환하고, 그렇지 않으면 false를 반환합니다.  
  
    -   [false 리터럴](../../../csharp/language-reference/keywords/false-literal.md) 부울 값 false를 나타냅니다.  
  
-   [stackalloc](../../../csharp/language-reference/keywords/stackalloc.md) 스택의 메모리 블록을 할당합니다.  
  
 연산자 및 문으로 사용할 수 있는 다음 키워드는 [문](../../../csharp/language-reference/keywords/statement-keywords.md) 섹션에서 설명합니다.  
  
-   [checked](../../../csharp/language-reference/keywords/checked.md) checked 컨텍스트를 지정합니다.  
  
-   [unchecked](../../../csharp/language-reference/keywords/unchecked.md) unchecked 컨텍스트를 지정합니다.
