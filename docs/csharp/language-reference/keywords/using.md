# <a name="using-c-reference"></a>using(C# 참조)
`using` 키워드의 주요 용도는 두 가지입니다.  
  
-   지시문으로서, 네임스페이스에 대한 별칭을 만들거나 다른 네임스페이스에 정의된 형식을 가져오는 데 사용됩니다. [using 지시문](../../../csharp/language-reference/keywords/using-directive.md)을 참조하세요.  
  
-   문으로서, 개체가 삭제될 끝에서 범위를 정의하는 데 사용됩니다. [using 문](../../../csharp/language-reference/keywords/using-statement.md)을 참조하세요.  
  
또한 [using static](using-static.md) 지시문을 사용하면 형식 이름을 지정하지 않고도 상태 멤버에 액세스할 수 있는 형식을 정의할 수 있습니다.
