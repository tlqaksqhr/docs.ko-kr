# 컨텍스트 키워드(C# 참조)
컨텍스트 키워드는 코드에서 특정 의미를 지정하는 데 사용되지만 C\#의 예약어는 아닙니다.  이 단원에서는 다음과 같은 컨텍스트 키워드를 설명합니다.  
  
|키워드|설명|  
|---------|--------|  
|[추가](../../../csharp/language-reference/keywords/add.md)|클라이언트 코드가 이벤트를 구독할 때 호출되는 사용자 지정 이벤트 접근자를 정의합니다.|  
|[async](../../../csharp/language-reference/keywords/async.md)|수정 된 메서드, 람다 식 또는 무명 메서드가 비동기입니다.|  
|[기다립니다](../../../csharp/language-reference/keywords/await.md)|바뀌게 된 작업이 완료 될 때까지 하는 비동기 메서드를 일시 중단 합니다.|  
|[dynamic](../../../csharp/language-reference/keywords/dynamic.md)|컴파일 타임 형식 검사를 무시하는 작업을 활성화하는 참조 형식을 정의합니다.|  
|[get](../../../csharp/language-reference/keywords/get.md)|속성이나 인덱서에 대한 접근자 메서드를 정의합니다.|  
|[global](../../../csharp/language-reference/keywords/global.md)|그렇지 않으면 명명되지 않는 기본 전역 네임스페이스를 지정합니다.|  
|[부분\(Partial\)](../../../csharp/language-reference/keywords/partial-type.md)|동일한 컴파일 단위 전체에서 partial 클래스, 구조체 및 인터페이스를 정의합니다.|  
|[remove](../../../csharp/language-reference/keywords/remove.md)|클라이언트 코드가 이벤트 구독을 취소할 때 호출되는 사용자 지정 이벤트 접근자를 정의합니다.|  
|[set](../../../csharp/language-reference/keywords/set.md)|속성이나 인덱서에 대한 접근자 메서드를 정의합니다.|  
|[value](../../../csharp/language-reference/keywords/value.md)|접근자를 설정하고 이벤트 처리기를 추가하거나 제거하는 데 사용됩니다.|  
|[var](../../../csharp/language-reference/keywords/var.md)|메서드 범위에서 선언되는 변수의 형식을 컴파일러에서 결정할 수 있게 합니다.|  
|[where](../../../csharp/language-reference/keywords/where-generic-type-constraint.md)|제네릭 선언에 제약 조건을 추가합니다.  자세한 내용은 [where](../../../csharp/language-reference/keywords/where-clause.md)를 참조하십시오.|  
|[yield](../../../csharp/language-reference/keywords/yield.md)|반복기 블록에서 열거자 개체에 값을 반환하거나 반복 종료를 알리는 데 사용됩니다.|  
  
 C\# 3.0에 도입된 쿼리 키워드도 모두 컨텍스트 키워드입니다.  자세한 내용은 [쿼리 키워드\(LINQ\)](../../../csharp/language-reference/keywords/query-keywords.md)를 참조하십시오.
