# <a name="contextual-keywords-c-reference"></a>상황별 키워드(C# 참조)
상황별 키워드는 코드에서 특정 의미를 제공하는 데 사용되지만 C#의 예약어입니다. 다음 상황별 키워드를 이 섹션에서 소개합니다.  
  
|키워드|설명|  
|-------------|-----------------|  
|[add](../../../csharp/language-reference/keywords/add.md)|클라이언트 코드가 이벤트를 구독할 때 호출되는 사용자 지정 이벤트 접근자를 정의합니다.|  
|[async](../../../csharp/language-reference/keywords/async.md)|수정된 메서드, 람다 식 또는 무명 메서드가 비동기임을 나타냅니다.|  
|[await](../../../csharp/language-reference/keywords/await.md)|대기 작업이 완료될 때까지 비동기 메서드를 일시 중단합니다.|  
|[dynamic](../../../csharp/language-reference/keywords/dynamic.md)|컴파일 시간 형식 검사를 우회하기 위해 수행하는 작업을 가능하게 하는 참조 형식을 정의합니다.|  
|[get](../../../csharp/language-reference/keywords/get.md)|속성 또는 인덱서에 대한 접근자 메서드를 정의합니다.|  
|[global](../../../csharp/language-reference/keywords/global.md)|별도로 명명되지 않은 기본 전역 네임스페이스를 지정합니다.|  
|[partial](../../../csharp/language-reference/keywords/partial-type.md)|같은 컴파일 단위 전체에서 partial 클래스, 구조체 및 인터페이스를 정의합니다.|  
|[remove](../../../csharp/language-reference/keywords/remove.md)|클라이언트 코드가 이벤트를 구독 취소할 때 호출되는 사용자 지정 이벤트 접근자를 정의합니다.|  
|[set](../../../csharp/language-reference/keywords/set.md)|속성 또는 인덱서에 대한 접근자 메서드를 정의합니다.|  
|[value](../../../csharp/language-reference/keywords/value.md)|접근자를 설정하고 이벤트 처리기를 추가하거나 제거하는 데 사용됩니다.|  
|[var](../../../csharp/language-reference/keywords/var.md)|컴파일러가 메서드 범위에 선언된 변수의 형식을 확인하는 데 사용됩니다.|  
|[when](when.md)|`switch` 문의 `catch` 블록 또는 `case` 레이블에 대한 필터 조건을 지정합니다.|
|[where](../../../csharp/language-reference/keywords/where-generic-type-constraint.md)|제네릭 선언에 제약 조건을 추가합니다. [where](../../../csharp/language-reference/keywords/where-clause.md)를 참조하세요.|  
|[yield](../../../csharp/language-reference/keywords/yield.md)|반복기 블록에서 값을 열거형 개체에 반환하거나 반복의 끝을 알리는 데 사용됩니다.|  
  
 C# 3.0에 도입된 모든 쿼리 키워드도 상황별 키워드입니다. 자세한 내용은 [쿼리 키워드(LINQ)](../../../csharp/language-reference/keywords/query-keywords.md)를 참조하세요.
