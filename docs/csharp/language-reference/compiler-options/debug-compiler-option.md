<span data-ttu-id="6b81b-103">**-debug** 옵션을 사용하면 컴파일러에서 디버깅 정보를 생성하여 출력 파일에 넣습니다.</span><span class="sxs-lookup"><span data-stu-id="6b81b-103">The **-debug** option causes the compiler to generate debugging information and place it in the output file or files.</span></span>

```console  
-debug[+ | -]  
-debug:{full | pdbonly}  
```  
  
## <a name="arguments"></a><span data-ttu-id="6b81b-105">인수</span><span class="sxs-lookup"><span data-stu-id="6b81b-105">Arguments</span></span>  
 <span data-ttu-id="6b81b-106">`+` &#124; `-`</span><span class="sxs-lookup"><span data-stu-id="6b81b-106">`+` &#124; `-`</span></span>  
 <span data-ttu-id="6b81b-107">`+` 또는 **-debug**를 지정하면 컴파일러에서 디버깅 정보를 생성하여 프로그램 데이터베이스(.pdb 파일)에 넣습니다.</span><span class="sxs-lookup"><span data-stu-id="6b81b-107">Specifying `+`, or just **-debug**, causes the compiler to generate debugging information and place it in a program database (.pdb file).</span></span> <span data-ttu-id="6b81b-108">`-`를 지정하면 **-debug**를 지정하지 않은 것으로 적용되어 디버그 정보가 생성되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="6b81b-108">Specifying `-`, which is in effect if you do not specify **-debug**, causes no debug information to be created.</span></span>  
  
 <span data-ttu-id="6b81b-109">`full` &#124; `pdbonly`</span><span class="sxs-lookup"><span data-stu-id="6b81b-109">`full` &#124; `pdbonly`</span></span>  
 <span data-ttu-id="6b81b-110">컴파일러에서 생성되는 디버깅 정보 형식을 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="6b81b-110">Specifies the type of debugging information generated by the compiler.</span></span> <span data-ttu-id="6b81b-111">**-debug:pdbonly**를 지정하지 않을 경우 적용되는 전체 인수를 통해 실행 중인 프로그램에 디버거를 연결할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="6b81b-111">The full argument, which is in effect if you do not specify **-debug:pdbonly**, enables attaching a debugger to the running program.</span></span> <span data-ttu-id="6b81b-112">pdbonly를 지정하면 디버거에서 프로그램이 시작되는 경우 소스 코드 디버깅이 가능하지만, 실행 중인 프로그램이 디버거에 연결되는 경우 어셈블러만 표시됩니다.</span><span class="sxs-lookup"><span data-stu-id="6b81b-112">Specifying pdbonly allows source code debugging when the program is started in the debugger but will only display assembler when the running program is attached to the debugger.</span></span>
