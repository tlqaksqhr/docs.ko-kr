# <a name="deployment-of-c-applications"></a>C# 응용 프로그램의 배포
C# 응용 프로그램 빌드를 완료한 후 다음 단계는 배포하는 것입니다. C#은 .NET 언어이므로 C# 실행 파일을 다른 컴퓨터에 배포하려면 .NET Framework를 각 실행 컴퓨터(및 가능할 경우 응용 프로그램에 관련된 기타 종속성)에 설치해야 합니다. .NET Framework 배포에 사용 가능한 다양한 옵션이 있습니다. 개요에 대해서는 [개발자를 위한 배포 가이드](https://msdn.microsoft.com/library/ee942965)를 참조하세요.  
  
 일반적으로 완료된 응용 프로그램을 다른 컴퓨터로 이동하는 작업을 배포라고 합니다. Microsoft 배포 환경에서는 배포 메커니즘을 제공합니다. 자세한 내용은 [Deploying Applications and Components](/visualstudio/deployment/deploying-applications-services-and-components)(응용 프로그램 및 구성 요소 배포)를 참조하세요.  
  
 주로 명령줄에서 빌드하고 배포할 경우 다른 배포 방법 및 종속성 재배포를 고려해야 합니다.
