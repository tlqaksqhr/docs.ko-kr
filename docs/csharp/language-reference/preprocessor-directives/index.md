컴파일러에는 별도의 전처리기가 없지만, 이 단원에 설명된 지시문은 전처리기가 있는 것처럼 처리됩니다. 지시문은 조건부 컴파일에서 지원하기 위해 사용됩니다. C 및 C++ 지시문과 달리, 매크로를 만들기 위해 이러한 지시문을 사용할 수 없습니다.

전처리기 지시문은 한 줄에서 유일한 명령이어야 합니다.
