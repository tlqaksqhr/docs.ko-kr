<span data-ttu-id="1b69c-103">'delegate' 대리자 형식에 out 매개 변수가 하나 이상 있으므로 매개 변수 목록이 없는 무명 메서드 블록을 이 대리자 형식으로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="1b69c-103">Cannot convert anonymous method block without a parameter list to delegate type 'delegate' because it has one or more out parameters</span></span>  
  
 <span data-ttu-id="1b69c-104">컴파일러가 대부분의 경우 무명 메서드 블록에서 매개 변수를 생략하는 것을 허용합니다.</span><span class="sxs-lookup"><span data-stu-id="1b69c-104">The compiler allows parameters to be omitted from an anonymous method block in most cases.</span></span> <span data-ttu-id="1b69c-105">무명 메서드 블록에 매개 변수 목록이 없지만 대리자에 `out` 매개 변수가 있는 경우 이 오류가 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="1b69c-105">This error arises when the anonymous method block does not have a parameter list, but the delegate has an `out` parameter.</span></span> <span data-ttu-id="1b69c-106">컴파일러가 `out` 매개 변수의 존재를 무시(이는 올바른 동작이 아닐 가능성이 높음)해야 하기 때문에 이 상황을 허용하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="1b69c-106">The compiler does not allow this situation because it would need to ignore the presence of the `out` parameter, which is unlikely to be the correct behavior.</span></span>
