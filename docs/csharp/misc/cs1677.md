# <a name="compiler-error-cs1677"></a><span data-ttu-id="cd6bd-102">컴파일러 오류 CS1677</span><span class="sxs-lookup"><span data-stu-id="cd6bd-102">Compiler Error CS1677</span></span>
<span data-ttu-id="cd6bd-103">'number' 매개 변수는 'keyword' 키워드를 사용하여 선언해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="cd6bd-103">Parameter 'number' should not be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="cd6bd-104">이 오류는 무명 메서드의 매개 변수 형식 한정자가 메서드를 캐스팅하는 대리자의 선언에서 사용되는 것과 일치하지 않는 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="cd6bd-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cd6bd-105">예</span><span class="sxs-lookup"><span data-stu-id="cd6bd-105">Example</span></span>  
 <span data-ttu-id="cd6bd-106">다음 샘플에서는 CS1677을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="cd6bd-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()   
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
