# <a name="compiler-error-cs1632"></a><span data-ttu-id="13acc-102">컴파일러 오류 CS1632</span><span class="sxs-lookup"><span data-stu-id="13acc-102">Compiler Error CS1632</span></span>
<span data-ttu-id="13acc-103">제어가 무명 메서드 또는 람다 식의 본문을 벗어날 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="13acc-103">Control cannot leave the body of an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="13acc-104">이 오류는 점프 문 발생 (**나누기**, `goto`, **계속**등)는 무명 메서드 블록 밖으로 제어를 이동 하려고 시도 합니다.</span><span class="sxs-lookup"><span data-stu-id="13acc-104">This error occurs if a jump statement (**break**, `goto`, **continue**, etc.) attempts to move control out of an anonymous method block.</span></span> <span data-ttu-id="13acc-105">무명 메서드 블록은 함수 본문이며 return 문이나 블록의 끝에 도달해야만 종료할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="13acc-105">An anonymous method block is a function body and can only be exited by a return statement or by reaching the end of the block.</span></span>  
  
 <span data-ttu-id="13acc-106">다음 샘플에서는 CS1632를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="13acc-106">The following sample generates CS1632:</span></span>  
  
```csharp  
// CS1632.cs  
// compile with: /target:library  
delegate void MyDelegate();  
class MyClass  
{  
   public void Test()  
   {        
      for (int i = 0 ; i < 5 ; i++)  
      {  
         MyDelegate d = delegate {  
            break;   // CS1632  
          };          
      }  
   }  
}  
```
