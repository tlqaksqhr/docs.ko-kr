 <span data-ttu-id="fc888-104">CS1909는 매개 변수 형식에 적용할 수 없는 `DefaultValue` 특성을 사용할 때 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="fc888-104">CS1909 is generated when you use a `DefaultValue` attribute that is not applicable on the parameter type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fc888-105">예제</span><span class="sxs-lookup"><span data-stu-id="fc888-105">Example</span></span>  
 <span data-ttu-id="fc888-106">다음 샘플에서는 CS1909를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fc888-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
