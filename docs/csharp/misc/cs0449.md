# <a name="compiler-error-cs0449"></a><span data-ttu-id="da7ae-102">컴파일러 오류 CS0449</span><span class="sxs-lookup"><span data-stu-id="da7ae-102">Compiler Error CS0449</span></span>
<span data-ttu-id="da7ae-103">'class' 또는 'struct' 제약 조건은 다른 모든 제약 조건보다 앞에 와야 합니다.</span><span class="sxs-lookup"><span data-stu-id="da7ae-103">The 'class' or 'struct' constraint must come before any other constraints</span></span>  
  
 <span data-ttu-id="da7ae-104">제네릭 형식 또는 메서드의 형식 매개 변수에 대한 제약 조건은 특정 순서로 발생해야 합니다. `class` 또는 `struct` 가 첫 번째여야 하고(있는 경우), 그다음에 인터페이스 제약 조건, 생성자 제약 조건 순으로 와야 합니다.</span><span class="sxs-lookup"><span data-stu-id="da7ae-104">The constraints on the type parameter of a generic type or method must occur in a specific order: `class` or `struct` must be first, if present, then any interface constraints, and finally any constructor constraints.</span></span> <span data-ttu-id="da7ae-105">이 오류는 `class` 또는 `struct` 제약 조건이 첫 번째로 표시되지 않을 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="da7ae-105">This error is caused by the `class` or `struct` constraint not appearing first.</span></span> <span data-ttu-id="da7ae-106">이 오류를 해결하려면 제약 조건 절을 다시 정렬합니다.</span><span class="sxs-lookup"><span data-stu-id="da7ae-106">To resolve this error, reorder the constraint clauses.</span></span>  
  
## <a name="example"></a><span data-ttu-id="da7ae-107">예</span><span class="sxs-lookup"><span data-stu-id="da7ae-107">Example</span></span>  
 <span data-ttu-id="da7ae-108">다음 샘플에서는 CS0449를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="da7ae-108">The following sample generates CS0449.</span></span>  
  
```csharp  
// CS0449.cs  
// compile with: /target:library  
interface I {}  
public class C4   
{  
   public void F1<T>() where T : class, struct, I {}   // CS0449  
   public void F2<T>() where T : I, struct {}   // CS0449  
   public void F3<T>() where T : I, class {}   // CS0449  
  
   // OK  
   public void F4<T>() where T : class {}  
   public void F5<T>() where T : struct {}  
   public void F6<T>() where T : I {}  
}  
```
