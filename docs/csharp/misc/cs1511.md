## <a name="example"></a><span data-ttu-id="830e9-106">예제</span><span class="sxs-lookup"><span data-stu-id="830e9-106">Example</span></span>  
 <span data-ttu-id="830e9-107">다음 샘플에서는 CS1511을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="830e9-107">The following sample generates CS1511.</span></span>  
  
```  
// CS1511.cs  
// compile with: /target:library  
public class A  
{  
   public int j = 0;  
}  
  
class C : A  
{  
   public void Method()  
   {  
      base.j = 3;   // base allowed here  
   }  
  
   public static int StaticMethod()  
   {  
      base.j = 3;   // CS1511  
      return 1;  
   }  
}  
```
