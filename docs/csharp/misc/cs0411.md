# <a name="compiler-error-cs0411"></a><span data-ttu-id="ee3ba-102">컴파일러 오류 CS0411</span><span class="sxs-lookup"><span data-stu-id="ee3ba-102">Compiler Error CS0411</span></span>
<span data-ttu-id="ee3ba-103">'method' 메서드의 형식 인수를 유추할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="ee3ba-103">The type arguments for method 'method' cannot be inferred from the usage.</span></span> <span data-ttu-id="ee3ba-104">형식 인수를 명시적으로 지정하세요.</span><span class="sxs-lookup"><span data-stu-id="ee3ba-104">Try specifying the type arguments explicitly.</span></span>  
  
 <span data-ttu-id="ee3ba-105">이 오류는 형식 인수를 명시적으로 제공하지 않고 제네릭 메서드를 호출하고 컴파일러에서 원하는 형식 인수를 유추할 수 없는 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="ee3ba-105">This error occurs if you call a generic method without explicitly providing the type arguments and the compiler cannot infer which type arguments are intended.</span></span> <span data-ttu-id="ee3ba-106">이 오류를 방지하려면 꺾쇠 괄호에 원하는 형식 인수를 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="ee3ba-106">To avoid this error, add the intended type arguments in angle brackets.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ee3ba-107">예제</span><span class="sxs-lookup"><span data-stu-id="ee3ba-107">Example</span></span>  
 <span data-ttu-id="ee3ba-108">다음 샘플에서는 CS0411을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ee3ba-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ee3ba-109">예제</span><span class="sxs-lookup"><span data-stu-id="ee3ba-109">Example</span></span>  
 <span data-ttu-id="ee3ba-110">기타 가능한 오류 사례에는 매개 변수가 `null`, 즉 형식 정보가 없는 경우가 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="ee3ba-110">Other possible error cases include when the parameter is `null`, which has no type information:</span></span>  
  
```csharp  
// CS0411b.cs  
class C  
{  
    public void F<T>(T t) where T : C   
    {  
    }  
  
    public static void Main()  
    {  
        C c = new C();  
        c.F(null);  // CS0411  
    }  
}  
```
