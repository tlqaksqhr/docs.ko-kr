 <span data-ttu-id="df414-106">다음 샘플에서는 CS0205를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="df414-106">The following sample generates CS0205:</span></span>

```csharp  
   abstract public void M();  
}

{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
