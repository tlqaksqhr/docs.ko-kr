## <a name="example"></a><span data-ttu-id="199f0-106">예</span><span class="sxs-lookup"><span data-stu-id="199f0-106">Example</span></span>

```csharp  
// CS3002.cs

public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
