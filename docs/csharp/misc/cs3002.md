## <a name="example"></a><span data-ttu-id="73510-106">예제</span><span class="sxs-lookup"><span data-stu-id="73510-106">Example</span></span>  
 <span data-ttu-id="73510-107">다음 예제에서는 CS3002를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="73510-107">The following example generates CS3002:</span></span>  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
