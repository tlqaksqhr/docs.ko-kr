## <a name="example"></a><span data-ttu-id="51aca-109">예</span><span class="sxs-lookup"><span data-stu-id="51aca-109">Example</span></span>  
 <span data-ttu-id="51aca-110">다음 코드에서는 CS1947을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="51aca-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
