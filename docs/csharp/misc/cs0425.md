# <a name="compiler-error-cs0425"></a><span data-ttu-id="77f41-102">컴파일러 오류 CS0425</span><span class="sxs-lookup"><span data-stu-id="77f41-102">Compiler Error CS0425</span></span>
<span data-ttu-id="77f41-103">'method' 메서드의 'type parameter' 형식 매개 변수에 대한 제약 조건이 'method' 인터페이스 메서드의 'type parameter' 형식 매개 변수에 대한 제약 조건과 일치해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="77f41-103">The constraints for type parameter 'type parameter' of method 'method' must match the constraints for type parameter 'type parameter' of interface method 'method'.</span></span> <span data-ttu-id="77f41-104">명시적 인터페이스 구현을 대신 사용하세요.</span><span class="sxs-lookup"><span data-stu-id="77f41-104">Consider using an explicit interface implementation instead.</span></span>  
  
 <span data-ttu-id="77f41-105">이 오류는 가상 제네릭 메서드가 파생 클래스에서 재정의되었으며 파생 클래스의 메서드에 대한 제약 조건이 기본 클래스의 메서드에 대한 제약 조건과 일치하지 않는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="77f41-105">This error occurs if a virtual generic method is overridden in a derived class and the constraints on the method in the derived class do not match the constraints on the method in the base class.</span></span> <span data-ttu-id="77f41-106">이 오류를 방지하려면 `where` 절이 두 선언에서 동일한지 확인하거나 인터페이스를 명시적으로 구현합니다.</span><span class="sxs-lookup"><span data-stu-id="77f41-106">To avoid this error, make sure the `where` clause is identical in both declarations, or implement the interface explicitly.</span></span>  
  
## <a name="example"></a><span data-ttu-id="77f41-107">예제</span><span class="sxs-lookup"><span data-stu-id="77f41-107">Example</span></span>  
 <span data-ttu-id="77f41-108">다음 예제에서는 CS0425를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="77f41-108">The following example generates CS0425:</span></span>  
  
```csharp  
// CS0425.cs  
  
class C1  
{  
}  
  
class C2  
{  
}  
  
interface IBase  
{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}  
  
class Derived : IBase  
{  
    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425  
    {  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
