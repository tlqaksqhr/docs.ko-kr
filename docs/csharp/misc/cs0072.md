# <a name="compiler-error-cs0072"></a><span data-ttu-id="5e386-102">컴파일러 오류 CS0072</span><span class="sxs-lookup"><span data-stu-id="5e386-102">Compiler Error CS0072</span></span>
<span data-ttu-id="5e386-103">'event': 재정의할 수 없습니다. 'method'가 이벤트가 아닙니다.</span><span class="sxs-lookup"><span data-stu-id="5e386-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="5e386-104">[event](../../csharp/language-reference/keywords/event.md) 는 다른 이벤트만 재정의할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="5e386-104">An [event](../../csharp/language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="5e386-105">자세한 내용은 [이벤트](../../csharp/programming-guide/events/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="5e386-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="5e386-106">다음 샘플에서는 CS0072를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5e386-106">The following sample generates CS0072:</span></span>  
  
```csharp  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
