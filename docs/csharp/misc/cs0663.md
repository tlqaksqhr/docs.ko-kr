 <span data-ttu-id="ad37a-104">매개 변수에서 [ref](../../csharp/language-reference/keywords/ref.md) 및 [out](../../csharp/language-reference/keywords/out.md) 사용만 다른 메서드는 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="ad37a-104">Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="ad37a-105">다음 샘플에서는 CS0663을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ad37a-105">The following sample generates CS0663:</span></span>  
  
```  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
