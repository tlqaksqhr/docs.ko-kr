# <a name="compiler-error-cs4009"></a><span data-ttu-id="288eb-102">컴파일러 오류 CS4009</span><span class="sxs-lookup"><span data-stu-id="288eb-102">Compiler Error CS4009</span></span>

<span data-ttu-id="288eb-103">'Type.Method': 진입점으로 표시할 수 없습니다는 `async` 한정자입니다.</span><span class="sxs-lookup"><span data-stu-id="288eb-103">'Type.Method': an entry point cannot be marked with the `async` modifier.</span></span>

<span data-ttu-id="288eb-104">사용할 수 없습니다는 `async` 응용 프로그램 진입점의 키워드 (일반적으로 `Main` 메서드).</span><span class="sxs-lookup"><span data-stu-id="288eb-104">You cannot use the `async` keyword in the application entry point (typically the `Main` method).</span></span>  

<span data-ttu-id="288eb-105">다음 예제에서는 CS4009을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="288eb-105">The following example produces CS4009:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```
