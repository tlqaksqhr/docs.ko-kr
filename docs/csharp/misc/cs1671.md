# <a name="compiler-error-cs1671"></a>컴파일러 오류 CS1671
네임스페이스 선언에는 한정자 또는 특성을 사용할 수 없습니다.  
  
 한정자는 네임스페이스에 적용할 때 의미가 없으므로 사용할 수 없습니다.  
  
 다음 샘플에서는 CS1671을 생성합니다.  
  
```  
// CS1671.cs  
public namespace NS // CS1671  
{  
  
}  
```
