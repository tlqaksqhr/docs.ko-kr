<span data-ttu-id="04242-103">'assembly'의 'namespace' 네임스페이스가 'assembly'에서 가져온 형식 'type'과 충돌합니다.</span><span class="sxs-lookup"><span data-stu-id="04242-103">The namespace 'namespace' in 'assembly' conflicts with the imported type 'type' in 'assembly'.</span></span> <span data-ttu-id="04242-104">'assembly'에 정의된 네임스페이스를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="04242-104">Using the namespace defined in 'assembly'..</span></span>  
  
 <span data-ttu-id="04242-105">이 경고는 원본 파일(file_2)의 네임스페이스가 file_1의 가져온 형식과 충돌할 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="04242-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="04242-106">컴파일러에서는 원본 파일에 있는 네임스페이스를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="04242-106">The compiler uses the one in the source file.</span></span>

```  
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="04242-109">그런 다음 아래 파일을 컴파일합니다.</span><span class="sxs-lookup"><span data-stu-id="04242-109">Then, compile this file:</span></span>

```  
   public class A { }  
}

      Console.WriteLine(typeof(Util.A)); // CS0435
```
