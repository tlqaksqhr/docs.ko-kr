## <a name="example"></a><span data-ttu-id="c0ee8-106">예제</span><span class="sxs-lookup"><span data-stu-id="c0ee8-106">Example</span></span>  
 <span data-ttu-id="c0ee8-107">다음 예제에서는 CS0471을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="c0ee8-107">The following example generates CS0471.</span></span>  
  
```  
// CS0471.cs  
// compile with: /t:library  
class Test  
{  
    public void F(bool x, bool y) {}  
    public void F1()  
    {  
        int a = 1, b = 2, c = 3;  
        F(a<b, c>(3));    // CS0471  
        // To resolve, try the following instead:  
        // F((a<b), c>(3));  
    }  
}  
```
