## <a name="example"></a><span data-ttu-id="e1ca4-106">예</span><span class="sxs-lookup"><span data-stu-id="e1ca4-106">Example</span></span>  
 <span data-ttu-id="e1ca4-107">다음 예제에서는 CS0082를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="e1ca4-107">The following example generates CS0082:</span></span>  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
