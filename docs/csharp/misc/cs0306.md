# <a name="compiler-error-cs0306"></a>컴파일러 오류 CS0306
'type' 형식은 형식 인수로 사용할 수 없습니다.  
  
 형식 매개 변수로 사용되는 형식이 허용되지 않습니다. 이는 형식이 포인터 형식이기 때문일 수 있습니다.  
  
 다음 예제에서는 CS0306을 생성합니다.  
  
```  
// CS0306.cs  
class C<T>  
{  
}  
  
class M  
{  
    // CS0306 – int* not allowed as a type parameter  
     C<int*> f;  
}  
```
