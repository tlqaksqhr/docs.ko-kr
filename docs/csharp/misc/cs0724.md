# <a name="compiler-error-cs0724"></a>컴파일러 오류 CS0724
어셈블리에 CLSCompliant 특성이 없으므로 CLSCompliant 특성이 필요하지 않습니다.  
  
 다음 예제에서는 `throw` 절 블록 내의 `finally` 문 때문에 CS0724를 생성합니다.  
  
## <a name="example"></a>예제  
 다음 예제에서는 CS0724를 생성합니다.  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
