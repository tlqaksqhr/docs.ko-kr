# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="113f5-102">컴파일러 경고(수준 2) CS3021</span><span class="sxs-lookup"><span data-stu-id="113f5-102">Compiler Warning (level 2) CS3021</span></span>
<span data-ttu-id="113f5-103">어셈블리에 CLSCompliant 특성이 없으므로 'type'에 CLSCompliant 특성이 필요하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="113f5-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="113f5-104">이 경고는 발생 하는 경우 `[CLSCompliant(false)]` 는 어셈블리 수준의 CLSCompliant 특성이 true로 설정 되지 않은 어셈블리의 클래스에 표시 됩니다 (즉, 줄 `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="113f5-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="113f5-105">어셈블리가 자체적으로 CLS 규격으로 선언하지 않아 비규격으로 간주되므로 어셈블리 내에서 별도로 비규격으로 선언할 필요가 없습니다.</span><span class="sxs-lookup"><span data-stu-id="113f5-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="113f5-106">CLS 규격에 대한 자세한 내용은 [CLS 규격 코드 작성](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="113f5-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>  
  
 <span data-ttu-id="113f5-107">이 경고가 발생하지 않게 하려면 특성을 제거하거나 어셈블리 수준 특성을 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="113f5-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="113f5-108">예제</span><span class="sxs-lookup"><span data-stu-id="113f5-108">Example</span></span>  
 <span data-ttu-id="113f5-109">다음 예제는 CS3021을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="113f5-109">The following example generates CS3021:</span></span>  
  
```  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
