# <a name="compiler-error-cs2018"></a>컴파일러 오류 CS2018
'cscmsgs.dll' 메시지 파일을 찾을 수 없습니다.  
  
 컴파일러의 오류 및 경고 메시지를 포함하는 .dll 파일을 찾을 수 없습니다. 이 파일은 다른 컴파일러 지원 파일과 동일한 디렉터리에 있어야 합니다.
