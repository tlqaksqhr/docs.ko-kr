# <a name="compiler-error-cs1619"></a>컴파일러 오류 CS1619
'filename' 임시 파일을 만들 수 없습니다. reason  
  
 지정된 이유로 컴파일러에서 임시 파일을 만들 수 없습니다(예: 디스크가 가득 참).
