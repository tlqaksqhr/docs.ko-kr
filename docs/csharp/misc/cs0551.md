## <a name="example"></a><span data-ttu-id="dffad-106">예</span><span class="sxs-lookup"><span data-stu-id="dffad-106">Example</span></span>  
 <span data-ttu-id="dffad-107">다음 샘플에서는 CS0551을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="dffad-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
