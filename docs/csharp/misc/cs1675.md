# <a name="compiler-error-cs1675"></a>컴파일러 오류 CS1675
열거형에는 형식 매개 변수를 사용할 수 없습니다.  
  
 이 오류를 해결하려면 `enum` 선언에서 형식 매개 변수를 제거합니다.  
  
## <a name="example"></a>예제  
 다음 샘플에서는 CS1675를 생성합니다.  
  
```csharp  
// CS1675.cs  
enum E<T>  // CS1675  
{  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
