# <a name="compiler-error-cs0637"></a><span data-ttu-id="b5352-102">컴파일러 오류 CS0637</span><span class="sxs-lookup"><span data-stu-id="b5352-102">Compiler Error CS0637</span></span>

<span data-ttu-id="b5352-103">Static 또는 const 필드에 FieldOffset 특성이 허용 되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="b5352-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="b5352-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 특성은 [정적](../../csharp/language-reference/keywords/static.md) 또는 [const](../../csharp/language-reference/keywords/const.md)로 표시된 필드에 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="b5352-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="b5352-105">다음 샘플에서는 CS0637을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="b5352-105">The following sample generates CS0637:</span></span>  
  
```  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
