## <a name="example"></a><span data-ttu-id="05492-107">예</span><span class="sxs-lookup"><span data-stu-id="05492-107">Example</span></span>  
 <span data-ttu-id="05492-108">다음 코드는 CS1952를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="05492-108">The following code produces CS1952:</span></span>

using System.Linq.Expressions;  
  
class Test  
{  
    public static int M(__arglist)

        return 1;  
    }

        Expression<Func<int, int>> f = x => Test.M(__arglist(x)); // CS1952

    }
