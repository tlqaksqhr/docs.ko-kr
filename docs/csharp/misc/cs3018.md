## <a name="example"></a><span data-ttu-id="c3b7e-108">예</span><span class="sxs-lookup"><span data-stu-id="c3b7e-108">Example</span></span>  
 <span data-ttu-id="c3b7e-109">다음 샘플에서는 CS3018을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="c3b7e-109">The following sample generates CS3018.</span></span>  
  
```  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
