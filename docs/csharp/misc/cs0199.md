# <a name="compiler-error-cs0199"></a><span data-ttu-id="9b6c5-102">컴파일러 오류 CS0199</span><span class="sxs-lookup"><span data-stu-id="9b6c5-102">Compiler Error CS0199</span></span>
<span data-ttu-id="9b6c5-103">정적 읽기 전용 필드 'name'의 필드는 ref 또는 out으로 전달할 수 없습니다. 단 정적 생성자에서는 예외입니다.</span><span class="sxs-lookup"><span data-stu-id="9b6c5-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="9b6c5-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) 변수에 동일한 있어야 [정적](../../csharp/language-reference/keywords/static.md) 변수로 전달 하려는 생성자와 사용은 [ref](../../csharp/language-reference/keywords/ref.md) 또는 [아웃](../../csharp/language-reference/keywords/out-parameter-modifier.md) 매개 변수입니다.</span><span class="sxs-lookup"><span data-stu-id="9b6c5-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="9b6c5-105">자세한 내용은 [매개 변수 전달](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="9b6c5-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="9b6c5-106">예제</span><span class="sxs-lookup"><span data-stu-id="9b6c5-106">Example</span></span>  
 <span data-ttu-id="9b6c5-107">다음 샘플에서는 CS0199를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="9b6c5-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
