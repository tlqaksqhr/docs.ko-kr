 <span data-ttu-id="78045-104">클래스 멤버가 [extern](../../csharp/language-reference/keywords/extern.md)으로 표시되는 경우 해당 멤버의 정의가 다른 파일에 있다는 것입니다.</span><span class="sxs-lookup"><span data-stu-id="78045-104">When a class member is marked [extern](../../csharp/language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="78045-105">따라서 **extern** 으로 표시된 클래스 멤버는 클래스에서 정의할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="78045-105">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="78045-106">`extern` 키워드를 제거하거나 정의를 삭제합니다.</span><span class="sxs-lookup"><span data-stu-id="78045-106">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="78045-107">자세한 내용은 [메서드](../../csharp/programming-guide/classes-and-structs/methods.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="78045-107">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>

   }
