# <a name="compiler-error-cs0529"></a><span data-ttu-id="70060-102">컴파일러 오류 CS0529</span><span class="sxs-lookup"><span data-stu-id="70060-102">Compiler Error CS0529</span></span>
<span data-ttu-id="70060-103">상속된 'interface1' 인터페이스는 'interface2'의 인터페이스 계층 구조에서 순환됩니다.</span><span class="sxs-lookup"><span data-stu-id="70060-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="70060-104">[interface](../../csharp/language-reference/keywords/interface.md) 의 상속 목록에 자신에 대한 직접 또는 간접 참조가 포함되어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="70060-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="70060-105">인터페이스는 다른 인터페이스에서만 상속할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="70060-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="70060-106">다음 샘플에서는 CS0529를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="70060-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
