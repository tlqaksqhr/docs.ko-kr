# <a name="compiler-error-cs1526"></a><span data-ttu-id="d1ff2-102">컴파일러 오류 CS1526</span><span class="sxs-lookup"><span data-stu-id="d1ff2-102">Compiler Error CS1526</span></span>
<span data-ttu-id="d1ff2-103">새 식에는 (),, 필요한 또는 {} 형식의 이름을 따서</span><span class="sxs-lookup"><span data-stu-id="d1ff2-103">A new expression requires (), [], or {} after type</span></span>  
  
 <span data-ttu-id="d1ff2-104">개체에 대한 메모리를 동적으로 할당하는 데 사용하는 [new](../../csharp/language-reference/keywords/new.md) 연산자를 올바르게 지정하지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="d1ff2-104">The [new](../../csharp/language-reference/keywords/new.md) operator, used to dynamically allocate memory for an object, was not specified correctly.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d1ff2-105">예제</span><span class="sxs-lookup"><span data-stu-id="d1ff2-105">Example</span></span>  
 <span data-ttu-id="d1ff2-106">다음 샘플은 `new` 를 사용하여 배열 및 개체에 대한 공간을 할당하는 방법을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="d1ff2-106">The following sample shows how to use `new` to allocate space for an array and an object.</span></span>  
  
```csharp  
// CS1526.cs  
public class y  
{  
   public static int i = 0;  
   public int myi = 0;  
}  
  
public class z  
{  
   public static void Main()  
   {  
      y py = new y;   // CS1526  
      y[] aoys = new y[10];   // Array of Ys  
  
      for (int i = 0; i < aoys.Length; i++)  
         aoys[i] = new y();   // an object of type y  
   }  
}  
```
