# <a name="compiler-error-cs0405"></a>컴파일러 오류 CS0405
'type parameter' 형식 매개 변수에 대한 'constraint' 제약 조건이 중복됩니다.  
  
 제네릭 선언에 대한 두 제약 조건이 동일합니다. 오류를 제거하려면 중복을 제거합니다.  
  
 다음 샘플에서는 CS0405를 생성합니다.  
  
```csharp  
// CS0405.cs  
interface I  
{  
}  
  
class C<T> where T : I, I  // CS0405.cs  
{  
}  
```
