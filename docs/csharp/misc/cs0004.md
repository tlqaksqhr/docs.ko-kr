# <a name="compiler-error-cs0004"></a>컴파일러 오류 CS0004
경고가 오류로 처리되었습니다.  
  
 `/warnaserror` 컴파일러 옵션으로 컴파일했습니다. 이렇게 하면 컴파일러가 경고를 오류로 생성합니다.
