# <a name="compiler-error-cs1648"></a><span data-ttu-id="12deb-102">컴파일러 오류 CS1648</span><span class="sxs-lookup"><span data-stu-id="12deb-102">Compiler Error CS1648</span></span>
<span data-ttu-id="12deb-103">읽기 전용 필드 'identifier'의 멤버는 수정할 수 없습니다. 단 생성자 또는 변수 이니셜라이저에서는 예외입니다.</span><span class="sxs-lookup"><span data-stu-id="12deb-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="12deb-104">이 오류는 수정할 수 없는 읽기 전용 필드의 멤버를 수정하려고 시도할 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="12deb-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="12deb-105">이 오류를 해결하려면 읽기 전용 필드에 대한 할당을 생성자 또는 변수 이니셜라이저로 제한하거나 필드의 선언에서 읽기 전용 키워드를 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="12deb-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>  
  
 <span data-ttu-id="12deb-106">다음 샘플에서는 CS1648을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="12deb-106">The following sample generates CS1648:</span></span>  
  
```  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
