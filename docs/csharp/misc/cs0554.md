# <a name="compiler-error-cs0554"></a><span data-ttu-id="07eb7-102">컴파일러 오류 CS0554</span><span class="sxs-lookup"><span data-stu-id="07eb7-102">Compiler Error CS0554</span></span>
<span data-ttu-id="07eb7-103">'conversion routine': 사용자가 파생 클래스로/에서의 변환을 정의했습니다.</span><span class="sxs-lookup"><span data-stu-id="07eb7-103">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="07eb7-104">파생 클래스 값으로의 사용자 정의 변환은 허용되지 않습니다. 이러한 연산자는 필요하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="07eb7-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="07eb7-105">사용자 정의 변환에 대한 자세한 내용은 C# 언어 사양의 6장을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="07eb7-105">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="07eb7-106">다음 샘플에서는 CS0554를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="07eb7-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
