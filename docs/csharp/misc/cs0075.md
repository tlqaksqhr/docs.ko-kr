# <a name="compiler-error-cs0075"></a><span data-ttu-id="03872-102">컴파일러 오류 CS0075</span><span class="sxs-lookup"><span data-stu-id="03872-102">Compiler Error CS0075</span></span>
<span data-ttu-id="03872-103">음의 값을 캐스팅하려면 값을 괄호로 묶어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="03872-103">To cast a negative value, you must enclose the value in parentheses</span></span>  
  
 <span data-ttu-id="03872-104">미리 정의된 형식을 식별하는 키워드를 사용하여 캐스팅하는 경우 괄호가 필요하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="03872-104">If you are casting using a keyword that identifies a predefined type, then you do not need parentheses.</span></span> <span data-ttu-id="03872-105">그렇지 않은 경우 (x)-y가 캐스트 식으로 고려되지 않기 때문에 괄호를 추가해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="03872-105">Otherwise, you must put the parentheses because (x) –y will not be considered a cast expression.</span></span> <span data-ttu-id="03872-106">C# 사양에서 7.6.6 섹션:</span><span class="sxs-lookup"><span data-stu-id="03872-106">From the C# Specification, Section 7.6.6:</span></span>  
  
 <span data-ttu-id="03872-107">*하는 경우 뒤는 명확성 규칙에서 x 및 y는 (x) y, 식별자 (마찬가지 및 (캐스트 식이 캐스트 식 이지만 (x)-y가 아니면 x가 형식을 식별 하는 경우에 합니다. 그러나 x가 미리 정의 된 형식 (예: int)를 식별 하는 키워드, 다음 4 가지 형태가 모두 캐스트 식 (이러한 키워드 될 수 없으므로 식을 단독으로).*</span><span class="sxs-lookup"><span data-stu-id="03872-107">*From the disambiguation rule it follows that, if x and y are identifiers, (x)y, (x)(y), and (x)(-y) are cast-expressions, but (x)-y is not, even if x identifies a type. However, if x is a keyword that identifies a predefined type (such as int), then all four forms are cast-expressions (because such a keyword could not possibly be an expression by itself).*</span></span>  
  
 <span data-ttu-id="03872-108">다음 코드에서는 CS0075를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="03872-108">The following code generates CS0075:</span></span>  
  
```csharp  
// CS0075  
namespace MyNamespace  
{  
    enum MyEnum { }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            // To fix the error, place the negative  
            // values below in parentheses  
            int i = (System.Int32) - 4; //CS0075  
            MyEnum e = (MyEnum) - 1;    //CS0075  
            System.Console.WriteLine(i); //to avoid warning  
            System.Console.WriteLine(e); //to avoid warning  
        }  
    }  
}  
```
