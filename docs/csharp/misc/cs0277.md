## <a name="example"></a><span data-ttu-id="4389d-108">예제</span><span class="sxs-lookup"><span data-stu-id="4389d-108">Example</span></span>  
 <span data-ttu-id="4389d-109">다음 예제는 CS0277을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4389d-109">The following example generates CS0277:</span></span>  
  
```csharp

    int Property  
    {  
        get;  
        set;  
    }

{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
