# <a name="compiler-error-cs0764"></a><span data-ttu-id="7a4db-102">컴파일러 오류 CS0764</span><span class="sxs-lookup"><span data-stu-id="7a4db-102">Compiler Error CS0764</span></span>
<span data-ttu-id="7a4db-103">두 부분 메서드(Partial Method) 선언 모두 unsafe이거나 unsafe가 아니어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="7a4db-103">Both partial method declarations must be unsafe or neither may be unsafe</span></span>  
  
 <span data-ttu-id="7a4db-104">부분 메서드(Partial Method)는 하나의 정의 선언(서명)과 선택적 구현 선언(본문)으로 구성됩니다.</span><span class="sxs-lookup"><span data-stu-id="7a4db-104">A partial method consists of a defining declaration (signature) and an optional implementing declaration (body).</span></span> <span data-ttu-id="7a4db-105">정의 선언에 `unsafe` 한정자가 있을 경우 구현 선언에도 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="7a4db-105">If the defining declaration has the `unsafe` modifier, the implementing declaration must also have it.</span></span> <span data-ttu-id="7a4db-106">반대로, 구현 선언에 `unsafe` 한정자가 있을 경우 정의 선언에도 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="7a4db-106">Conversely, if the implementing declaration has the `unsafe` modifier, the defining declaration must also.</span></span>
