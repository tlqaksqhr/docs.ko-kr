# <a name="compiler-error-cs0546"></a><span data-ttu-id="928f4-102">컴파일러 오류 CS0546</span><span class="sxs-lookup"><span data-stu-id="928f4-102">Compiler Error CS0546</span></span>
<span data-ttu-id="928f4-103">'accessor': 'property'에 재정의 가능한 set 접근자가 없으므로 재정의할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="928f4-103">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="928f4-104">접근자를 재정의할 수 없으므로 속성에 대한 접근자 메서드 중 하나를 재정의하지 못했습니다.</span><span class="sxs-lookup"><span data-stu-id="928f4-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="928f4-105">이 오류는 다음과 같은 경우에 발생할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="928f4-105">This error can occur if:</span></span>  
  
-   <span data-ttu-id="928f4-106">기본 클래스 속성이 [가상](../../csharp/language-reference/keywords/virtual.md)으로 선언되지 않았습니다.</span><span class="sxs-lookup"><span data-stu-id="928f4-106">the base class property is not declared as [virtual](../../csharp/language-reference/keywords/virtual.md).</span></span>  
  
-   <span data-ttu-id="928f4-107">기본 클래스 속성이 재정의하려는 [get](../../csharp/language-reference/keywords/get.md) 또는 [set](../../csharp/language-reference/keywords/set.md) 접근자를 선언하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="928f4-107">the base class property does not declare the [get](../../csharp/language-reference/keywords/get.md) or [set](../../csharp/language-reference/keywords/set.md) accessor you are trying to override.</span></span>  
  
 <span data-ttu-id="928f4-108">기본 클래스 속성을 재정의하지 않으려는 경우 파생 클래스에서 속성 앞에 [new](../../csharp/language-reference/keywords/new.md) 키워드를 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="928f4-108">If you do not want to override the base class property, you can use the [new](../../csharp/language-reference/keywords/new.md) keyword before the property in derived class.</span></span>  
  
 <span data-ttu-id="928f4-109">자세한 내용은 [속성 사용](../../csharp/programming-guide/classes-and-structs/using-properties.md)을 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="928f4-109">For more information, see [Using Properties](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="928f4-110">예</span><span class="sxs-lookup"><span data-stu-id="928f4-110">Example</span></span>  
 <span data-ttu-id="928f4-111">다음 샘플에서는 기본 클래스가 속성에 대한 set 접근자를 선언하지 않으므로 CS0546을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="928f4-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
