# <a name="compiler-error-cs0431"></a><span data-ttu-id="844ad-102">컴파일러 오류 CS0431</span><span class="sxs-lookup"><span data-stu-id="844ad-102">Compiler Error CS0431</span></span>
<span data-ttu-id="844ad-103">'identifier' 별칭은 형식을 참조하므로 '::'과 함께 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="844ad-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="844ad-104">대신 '.'를 사용하세요.</span><span class="sxs-lookup"><span data-stu-id="844ad-104">Use '.' instead.</span></span>  
  
 <span data-ttu-id="844ad-105">형식을 참조하는 별칭과 함께 "::"을 사용했습니다.</span><span class="sxs-lookup"><span data-stu-id="844ad-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="844ad-106">이 오류를 해결하려면 "." 연산자를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="844ad-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="844ad-107">다음 샘플에서는 CS0431을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="844ad-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
