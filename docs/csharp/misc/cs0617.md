# <a name="compiler-error-cs0617"></a><span data-ttu-id="a40a3-102">컴파일러 오류 CS0617</span><span class="sxs-lookup"><span data-stu-id="a40a3-102">Compiler Error CS0617</span></span>
<span data-ttu-id="a40a3-103">'reference'는 올바른 특성 매개 변수 형식이 아니므로 올바로 명명된 특성 인수가 아닙니다.</span><span class="sxs-lookup"><span data-stu-id="a40a3-103">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="a40a3-104">특성 클래스의 [private](../../csharp/language-reference/keywords/private.md) 멤버에 액세스하려고 했습니다.</span><span class="sxs-lookup"><span data-stu-id="a40a3-104">An attempt was made to access a [private](../../csharp/language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a40a3-105">예제</span><span class="sxs-lookup"><span data-stu-id="a40a3-105">Example</span></span>  
 <span data-ttu-id="a40a3-106">다음 샘플에서는 CS0617을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a40a3-106">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |   
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
