# <a name="compiler-warning-level-4-cs1712"></a><span data-ttu-id="f37a1-102">컴파일러 경고(수준 4) CS1712</span><span class="sxs-lookup"><span data-stu-id="f37a1-102">Compiler Warning (level 4) CS1712</span></span>
<span data-ttu-id="f37a1-103">'type'의 XML 주석에 'type parameter' 형식 매개 변수와 짝이 맞는 형식 매개 변수 태그가 없습니다. 다른 형식 매개 변수는 짝이 맞는 태그가 있습니다.</span><span class="sxs-lookup"><span data-stu-id="f37a1-103">Type parameter 'type parameter' has no matching typeparam tag in the XML comment on 'type' (but other type parameters do)</span></span>  
  
 <span data-ttu-id="f37a1-104">제네릭 형식의 설명서에 **typeparam** 태그가 없습니다.</span><span class="sxs-lookup"><span data-stu-id="f37a1-104">The documentation of a generic type is missing a **typeparam** tag.</span></span> <span data-ttu-id="f37a1-105">자세한 내용은 참조 [ \<typeparam >](../../csharp/programming-guide/xmldoc/typeparam.md)합니다.</span><span class="sxs-lookup"><span data-stu-id="f37a1-105">For more information, see [\<typeparam>](../../csharp/programming-guide/xmldoc/typeparam.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="f37a1-106">예제</span><span class="sxs-lookup"><span data-stu-id="f37a1-106">Example</span></span>  
 <span data-ttu-id="f37a1-107">다음 코드에서는 CS1712 경고를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="f37a1-107">The following code generates warning CS1712.</span></span> <span data-ttu-id="f37a1-108">이 오류를 해결하려면 형식 매개 변수 S에 대한 **typeparam** 태그를 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="f37a1-108">To resolve this error, add a **typeparam** tag for the type parameter S.</span></span>  
  
```csharp  
// CS1712.cs  
// compile with: /doc:cs1712.xml  
using System;  
class Test  
{  
    public static void Main() {}  
    /// <param name="j"> This is the j parameter.</param>  
    /// <typeparam name="T"> This is the T type parameter.</typeparam>  
    public void bar<T,S>(int j) {}  // CS1712  
}  
```
