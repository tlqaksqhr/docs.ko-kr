## <a name="example"></a><span data-ttu-id="fda38-108">예</span><span class="sxs-lookup"><span data-stu-id="fda38-108">Example</span></span>

```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }

    {  
    }

    {  
        static void Main()  
        {

    }
```  
