 <span data-ttu-id="5fa5b-104">메서드가 대리자 형식과 호환되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="5fa5b-104">The method was not compatible with the delegate type.</span></span> <span data-ttu-id="5fa5b-105">인수 형식은 일치하지만 반환 형식이 해당 대리자의 올바른 반환 형식이 아닙니다.</span><span class="sxs-lookup"><span data-stu-id="5fa5b-105">The argument types matched, but the return type was not the correct return type for that delegate.</span></span> <span data-ttu-id="5fa5b-106">이 오류를 방지하려면 다른 메서드를 사용하여 메서드의 반환 형식을 변경하거나 대리자의 반환 형식을 변경합니다.</span><span class="sxs-lookup"><span data-stu-id="5fa5b-106">To avoid this error, use a different method, change the method's return type, or change the delegate's return type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5fa5b-107">예제</span><span class="sxs-lookup"><span data-stu-id="5fa5b-107">Example</span></span>  
 <span data-ttu-id="5fa5b-108">다음 샘플에서는 CS0407을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="5fa5b-108">The following sample generates CS0407:</span></span>  
  
```csharp  
// CS0407.cs  
public delegate int MyDelegate();  
  
class C  
{  
    MyDelegate d;  
  
    public C()  
    {  
        d = new MyDelegate(F);  // OK: F returns int  
        d = new MyDelegate(G);  // CS0407 – G doesn't return int  
    }  
  
    public int F()  
    {  
        return 1;  
    }  
  
    public void G()  
    {  
    }  
  
    public static void Main()  
    {  
        C c1 = new C();  
    }  
}  
```
