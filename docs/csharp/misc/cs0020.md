# <a name="compiler-error-cs0020"></a>컴파일러 오류 CS0020
상수 0으로 나누었습니다.  
  
 식에서 나누기 연산의 분모에 리터럴(변수 아님) 값 0을 사용합니다. 0으로 나누기는 정의되지 않았으므로 사용할 수 없습니다.  
  
 다음 샘플에서는 CS0020을 생성합니다.  
  
```csharp  
// CS0020.cs  
namespace x  
{  
   public class b  
   {  
      public static void Main()  
      {  
         1 / 0;   // CS0020  
      }  
   }  
}  
```
