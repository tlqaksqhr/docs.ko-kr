# <a name="compiler-error-cs0575"></a>컴파일러 오류 CS0575
클래스 형식만 소멸자를 포함할 수 있습니다.  
  
 [struct](../../csharp/language-reference/keywords/struct.md) 는 소멸자를 포함할 수 없습니다.  
  
 다음 샘플에서는 CS0575를 생성합니다.  
  
```csharp  
// CS0575.cs  
namespace x  
{  
   public struct iii  
   {  
      ~iii()   // CS0575  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
