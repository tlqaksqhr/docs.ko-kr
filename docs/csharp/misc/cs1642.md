# <a name="compiler-error-cs1642"></a><span data-ttu-id="120d1-102">컴파일러 오류 CS1642</span><span class="sxs-lookup"><span data-stu-id="120d1-102">Compiler Error CS1642</span></span>
<span data-ttu-id="120d1-103">고정 크기 버퍼 필드는 구조체의 멤버로만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="120d1-103">Fixed size buffer fields may only be members of structs.</span></span>  
  
 <span data-ttu-id="120d1-104">이 오류는 `class`대신 `struct`의 고정 크기 버퍼 필드를 사용하는 경우에 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="120d1-104">This error occurs if you use a fixed size buffer field in a `class`, instead of a `struct`.</span></span> <span data-ttu-id="120d1-105">이 오류를 해결하려면 `class` 를 `struct` 로 변경하거나 필드를 일반 배열로 선언합니다.</span><span class="sxs-lookup"><span data-stu-id="120d1-105">To resolve this error, change the `class` to a `struct` or declare the field as an ordinary array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="120d1-106">예제</span><span class="sxs-lookup"><span data-stu-id="120d1-106">Example</span></span>  
 <span data-ttu-id="120d1-107">다음 샘플에서는 CS1642를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="120d1-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
