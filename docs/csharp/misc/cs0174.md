# <a name="compiler-error-cs0174"></a>컴파일러 오류 CS0174
'base' 참조에는 기본 클래스가 필요합니다.  
  
 이 오류는 .NET Framework 공용 언어 런타임이 기본 클래스가 없는 유일한 클래스인 `System.Object` 클래스에 대한 소스 코드를 컴파일하는 경우에만 발생할 수 있습니다.
