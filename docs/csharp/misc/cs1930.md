 <span data-ttu-id="03f0d-104">쿼리 식의 범위 변수는 쿼리 식이 종료될 때까지 범위 내에 있습니다.</span><span class="sxs-lookup"><span data-stu-id="03f0d-104">The range variable in a query expression is in scope until the query expression terminates.</span></span> <span data-ttu-id="03f0d-105">따라서 고유 식별자가 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="03f0d-105">It must therefore have a unique identifier.</span></span>

## <a name="example"></a><span data-ttu-id="03f0d-108">예제</span><span class="sxs-lookup"><span data-stu-id="03f0d-108">Example</span></span>  
 <span data-ttu-id="03f0d-109">다음 예제에서는 `num` 절의 범위 변수와 `from` 절에 사용된 범위 변수에 대해 `let` 식별자가 사용되기 때문에 CS1930을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="03f0d-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
