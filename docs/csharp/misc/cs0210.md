# <a name="compiler-error-cs0210"></a><span data-ttu-id="4818a-102">컴파일러 오류 CS0210</span><span class="sxs-lookup"><span data-stu-id="4818a-102">Compiler Error CS0210</span></span>
<span data-ttu-id="4818a-103">fixed 또는 using 문 선언에 이니셜라이저를 입력해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="4818a-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="4818a-104">[fixed 문](../../csharp/language-reference/keywords/fixed-statement.md)에서 변수를 선언하고 초기화해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="4818a-104">You must declare and initialize the variable in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="4818a-105">자세한 내용은 [안전하지 않은 코드 및 포인터](../../csharp/programming-guide/unsafe-code-pointers/index.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="4818a-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="4818a-106">다음 샘플에서는 CS0210을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4818a-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="4818a-107">또한 다음 샘플에서는 [using 문](../../csharp/language-reference/keywords/using-statement.md) 에 이니셜라이저가 없기 때문에 CS0210을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="4818a-107">The following sample also generates CS0210 because the [using statement](../../csharp/language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
