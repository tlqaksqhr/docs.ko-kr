 <span data-ttu-id="61a35-104">이 오류는 변수를 사용하고 해당 주소를 사용하려고 하며 무명 메서드 내에서 이러한 작업 중 하나를 수행하는 경우에 생성됩니다.</span><span class="sxs-lookup"><span data-stu-id="61a35-104">This error is generated when you use a variable, and attempt to take its address, and one of these actions is done inside an anonymous method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="61a35-105">예제</span><span class="sxs-lookup"><span data-stu-id="61a35-105">Example</span></span>  
 <span data-ttu-id="61a35-106">다음 샘플에서는 CS1686을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="61a35-106">The following sample generates CS1686.</span></span>  
  
```  
// CS1686.cs  
// compile with: /unsafe /target:library  
class MyClass  
{  
   public unsafe delegate int * MyDelegate();  
  
   public unsafe int * Test()  
   {  
      int j = 0;  
      MyDelegate d = delegate { return &j; };   // CS1686  
      return &j;   // OK  
   }  
}  
```
