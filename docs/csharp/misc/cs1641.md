 <span data-ttu-id="cee8c-104">정규 배열과 달리 고정 크기 버퍼에서는 선언 지점에서 상수 크기를 지정해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="cee8c-104">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="cee8c-105">이 오류를 해결하려면 양의 정수 리터럴 또는 상수 양의 정수를 추가하고 식별자 뒤에 대괄호를 넣습니다.</span><span class="sxs-lookup"><span data-stu-id="cee8c-105">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="cee8c-106">다음 샘플에서는 CS1641을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="cee8c-106">The following sample generates CS1641:</span></span>  
  
```csharp

   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
