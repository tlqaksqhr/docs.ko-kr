## <a name="example"></a><span data-ttu-id="d4fd1-110">예제</span><span class="sxs-lookup"><span data-stu-id="d4fd1-110">Example</span></span>  
 <span data-ttu-id="d4fd1-111">다음 코드에서는 CS0728 경고를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="d4fd1-111">The following code will generate warning CS0728.</span></span>  
  
```csharp 
// CS0728.cs  
  
using System;  
public class ValidBase : IDisposable  
{  
    public void Dispose() {  }  
}  
  
public class Logger  
{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)   
        {  
            vb = null;  // CS0728  
        }  
        vb = null;  
    }  
    public static void Main() { }  
}  
```
