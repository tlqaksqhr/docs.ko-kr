# <a name="compiler-error-cs0215"></a><span data-ttu-id="aa413-102">컴파일러 오류 CS0215</span><span class="sxs-lookup"><span data-stu-id="aa413-102">Compiler Error CS0215</span></span>
<span data-ttu-id="aa413-103">True 또는 False 연산자의 반환 형식은 bool이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="aa413-103">The return type of operator True or False must be bool</span></span>  
  
 <span data-ttu-id="aa413-104">사용자 정의 [true](../../csharp/language-reference/keywords/true.md) 및 [false](../../csharp/language-reference/keywords/false.md) 연산자에 [bool](../../csharp/language-reference/keywords/bool.md)의 반환 형식이 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="aa413-104">User-defined [true](../../csharp/language-reference/keywords/true.md) and [false](../../csharp/language-reference/keywords/false.md) operators must have a return type of [bool](../../csharp/language-reference/keywords/bool.md).</span></span> <span data-ttu-id="aa413-105">자세한 내용은 [오버로드할 수 있는 연산자](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="aa413-105">For more information, see [Overloadable Operators](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span></span>  
  
 <span data-ttu-id="aa413-106">다음 샘플에서는 CS0215를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="aa413-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
