<span data-ttu-id="15b67-103">++ 또는 -- 연산자의 반환 형식은 포함하는 형식이거나 포함하는 형식에서 파생되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="15b67-103">The return type for ++ or -- operator must be the containing type or derived from the containing type</span></span>

## <a name="example"></a><span data-ttu-id="15b67-105">예</span><span class="sxs-lookup"><span data-stu-id="15b67-105">Example</span></span>  
 <span data-ttu-id="15b67-106">다음 샘플에서는 CS0448을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="15b67-106">The following sample generates CS0448.</span></span>  
  
```csharp  
// CS0448.cs  
class C5  
{  
   public static int operator ++(C5 c) { return null; }   // CS0448  
   public static C5 operator --(C5 c) { return null; }   // OK  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="15b67-107">예</span><span class="sxs-lookup"><span data-stu-id="15b67-107">Example</span></span>

```csharp  
   public static T? operator ++(T? t) { return new T(); }  
  
   public static void Main() {}  
}  
```
