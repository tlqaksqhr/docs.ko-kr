# <a name="compiler-warning-level-2-cs0279"></a><span data-ttu-id="60148-102">컴파일러 경고(수준 2) CS0279</span><span class="sxs-lookup"><span data-stu-id="60148-102">Compiler Warning (level 2) CS0279</span></span>
<span data-ttu-id="60148-103">'type name'이 'pattern name' 패턴을 구현하지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="60148-103">'type name' does not implement the 'pattern name' pattern.</span></span> <span data-ttu-id="60148-104">'method name’이 static이거나 아니면 public이 아닙니다.</span><span class="sxs-lookup"><span data-stu-id="60148-104">'method name' is either static or not public.</span></span>  
  
 <span data-ttu-id="60148-105">C#에는 `foreach` 및 `using`과 같이 정의된 패턴을 사용하는 문이 여러 개 있습니다.</span><span class="sxs-lookup"><span data-stu-id="60148-105">There are several statements in C# that rely on defined patterns, such as `foreach` and `using`.</span></span> <span data-ttu-id="60148-106">예를 들어 `foreach` 는 열거 가능 패턴을 구현하는 컬렉션 클래스를 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="60148-106">For example, `foreach` relies on the collection class implementing the enumerable pattern.</span></span> <span data-ttu-id="60148-107">이 오류는 `static` 이 아닌 `public`으로 선언된 메서드로 인해 컴파일러가 일치시킬 수 없을 때 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="60148-107">This error occurs when the compiler is unable to make the match due to a method being declared `static` or not `public`.</span></span> <span data-ttu-id="60148-108">패턴의 메서드는 클래스의 인스턴스여야 하고 public이어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="60148-108">Methods in patterns are required to be instances of classes, and to be public.</span></span>  
  
## <a name="example"></a><span data-ttu-id="60148-109">예제</span><span class="sxs-lookup"><span data-stu-id="60148-109">Example</span></span>  
 <span data-ttu-id="60148-110">다음 예제에서는 CS0279를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="60148-110">The following example generates CS0279:</span></span>  
  
```csharp  
// CS0279.cs  
  
using System;  
using System.Collections;  
  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    internal IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    public static void Main()  
    {  
        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
