# <a name="for-further-reading-parallel-programming"></a>추가 정보(병렬 프로그래밍)
다음 리소스는 .NET Framework의 병렬 프로그래밍에 대한 추가 정보를 제공합니다.  
  
-   MSDN 웹 사이트의 [Parallel Computing Developer Center](http://go.microsoft.com/fwlink/?LinkID=160570)에는 .NET Framework 및 네이티브 C++에서의 병렬 프로그래밍에 대한 최신 내용과 포럼 게시물로 연결되는 링크가 있습니다.  
  
-   MSDN 블로그 사이트의 [Parallel Programming with .NET](http://go.microsoft.com/fwlink/?LinkID=169627)(.NET을 사용한 병렬 프로그래밍) 블로그에는 .NET Framework에서의 병렬 프로그래밍에 관해 심층적으로 다루는 여러 문서가 포함되어 있습니다.  
  
-   MSDN 블로그 사이트의 [Concurrency Visualizer(동시성 시각화 도우미) 블로그](http://go.microsoft.com/fwlink/?LinkID=169630)에서는 일부 버전의 [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)]에 포함된 새로운 성능 프로파일링 도구에 관해 다루고 있습니다.  
  
-   MSDN 포럼 사이트의 [병렬 확장 포럼](http://go.microsoft.com/fwlink/?LinkID=169628)에서는 병렬 프로그래밍과 관련된 내용을 질문하고 답변할 수 있습니다.  
  
-   MSDN 코드 갤러리 웹 사이트의 [Parallel Extensions Samples](http://go.microsoft.com/fwlink/?LinkID=165717)(병렬 확장 샘플) 페이지에서는 중급 및 고급 수준의 병렬 프로그래밍 기술을 보여주는 여러 가지 샘플을 제공합니다.
