 <span data-ttu-id="a5ae4-117">.NET 형식에서는 계층 구조를 의미하는 스키마의 이름을 지정하는데 점 구문을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="a5ae4-117">.NET types use a dot syntax naming scheme that connotes a hierarchy.</span></span> <span data-ttu-id="a5ae4-118">이 방법을 사용하면 관련 형식을 네임스페이스로 그룹화하여 보다 쉽게 검색하고 참조할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="a5ae4-118">This technique groups related types into namespaces so they can be searched and referenced more easily.</span></span> <span data-ttu-id="a5ae4-119">전체 이름에서 첫 번째 부분(가장 오른쪽 점의 오른쪽 부분)은 해당 네임스페이스의 이름이고,</span><span class="sxs-lookup"><span data-stu-id="a5ae4-119">The first part of the full name — up to the rightmost dot — is the namespace name.</span></span> <span data-ttu-id="a5ae4-120">마지막 부분은 형식 이름입니다.</span><span class="sxs-lookup"><span data-stu-id="a5ae4-120">The last part of the name is the type name.</span></span> <span data-ttu-id="a5ae4-121">예를 들어 **System.Collections.ArrayList**는 **System.Collections** 네임스페이스에 속하는 **ArrayList** 형식을 나타냅니다.</span><span class="sxs-lookup"><span data-stu-id="a5ae4-121">For example, **System.Collections.ArrayList** represents the **ArrayList** type, which belongs to the **System.Collections** namespace.</span></span> <span data-ttu-id="a5ae4-122">**System.Collections**의 형식은 개체의 컬렉션을 조작하는 데 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="a5ae4-122">The types in **System.Collections** can be used to manipulate collections of objects.</span></span>
