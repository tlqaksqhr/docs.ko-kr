>  <span data-ttu-id="ff035-109">문자 이스케이프는 정규식 패턴에서는 인식되지만 대체 패턴에서 인식되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="ff035-109">Character escapes are recognized in regular expression patterns but not in replacement patterns.</span></span>
