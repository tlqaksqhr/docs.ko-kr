# <a name="extending-xslt-style-sheets"></a>XSLT 스타일시트 확장
이 단원에서는 XSLT 기능을 확장하는 여러 가지 방법을 설명합니다. <xref:System.Xml.Xsl.XsltArgumentList> 클래스를 사용하여 확장 개체 또는 매개 변수를 추가할 수 있습니다. 그런 다음 스타일시트에서 확장명 개체 및 매개 변수를 호출할 수 있습니다. 또한 `msxsl:script` 요소를 사용하여 스크립트 블록을 스타일시트에 포함할 수도 있습니다.
