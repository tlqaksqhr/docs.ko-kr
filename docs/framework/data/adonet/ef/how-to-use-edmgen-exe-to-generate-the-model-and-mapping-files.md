### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="427cc-116">EdmGen.exe를 사용하여 Visual Basic 프로젝트용 School 모델을 생성하려면</span><span class="sxs-lookup"><span data-stu-id="427cc-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>  
  
1.  <span data-ttu-id="427cc-117">School 데이터베이스를 만듭니다.</span><span class="sxs-lookup"><span data-stu-id="427cc-117">Create the School database.</span></span> <span data-ttu-id="427cc-118">자세한 내용은 참조 [School 샘플 데이터베이스 만들기](http://msdn.microsoft.com/library/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0)합니다.</span><span class="sxs-lookup"><span data-stu-id="427cc-118">For more information, see [Creating the School Sample Database](http://msdn.microsoft.com/library/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0).</span></span>  
  
2.  <span data-ttu-id="427cc-119">명령 프롬프트에서 줄 바꿈 없이 다음 명령을 실행합니다.</span><span class="sxs-lookup"><span data-stu-id="427cc-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
