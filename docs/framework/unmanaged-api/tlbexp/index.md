[형식 라이브러리 내보내기 도구](../../../../docs/framework/tools/tlbexp-exe-type-library-exporter.md) (Tlbexp.exe) TlbRef.dll 라는 동적 연결 라이브러리를 로드 합니다. 이 DLL 두 도우미 함수 및 어셈블리를 형식 라이브러리로 변환 하는 동안 내보내기 도구를 사용 하는 인터페이스를 포함 합니다.
