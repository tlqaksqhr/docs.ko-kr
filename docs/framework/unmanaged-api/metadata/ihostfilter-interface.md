# <a name="ihostfilter-interface"></a>IHostFilter 인터페이스
지정된 된 토큰은 처리할 수를 지정 하는 메서드를 제공 합니다.  
  
> [!NOTE]
>  `IHostFilter` 클라이언트에서 인터페이스를 구현 해야 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[MarkToken 메서드](../../../../docs/framework/unmanaged-api/metadata/ihostfilter-marktoken-method.md)|지정한 메타 데이터 토큰이 처리할지를 나타냅니다.|
