# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA 메서드
지정한 토큰이 참조 하는 필드의 상대 가상 주소에 대 한 전역 변수 값을 설정 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT SetFieldRVA (   
    [in]  mdFieldDef  fd,   
    [in]  ULONG       ulRVA   
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `fd`  
 [in] 대상 필드에 대 한 토큰입니다.  
  
 `ulRVA`  
 [in] 주소는 코드 또는 데이터 영역입니다.
