# <a name="imetadataimportcloseenum-method"></a>IMetaDataImport::CloseEnum 메서드
지정 된 핸들에 의해 식별 되는 열거자를 닫습니다.  
  
## <a name="syntax"></a>구문  
  
```  
void CloseEnum (  
   [in] HCORENUM hEnum  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `hEnum`  
 [in] 닫습니다 열거자에 대 한 핸들입니다.
