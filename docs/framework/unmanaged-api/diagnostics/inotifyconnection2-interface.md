# <a name="inotifyconnection2-interface"></a>INotifyConnection2 인터페이스
등록 및 연결 알림 소스를 등록 취소 하기 위한 메서드를 선언 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[RegisterNotifySource 메서드](../../../../docs/framework/unmanaged-api/diagnostics/inotifyconnection2-registernotifysource-method.md)|지정 된 알림 원본 개체를 설치합니다.|  
|[UnregisterNotifySource 메서드](../../../../docs/framework/unmanaged-api/diagnostics/inotifyconnection2-unregisternotifysource-method.md)|연결에서 지정 된 알림 원본 개체를 제거합니다.|
