# <a name="icordebuggenericvaluegetvalue-method"></a>ICorDebugGenericValue::GetValue 메서드
지정된 된 버퍼에이 제네릭의 값을 복사합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetValue (  
    [out] void     *pTo  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pTo`  
 [out] 이 ICorDebugGenericValue 개체로 표시 하는 값에 대 한 포인터입니다. 값은 단순 형식 또는 참조 형식 (즉, 포인터) 수 있습니다.
