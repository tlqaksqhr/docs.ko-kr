# <a name="icordebugenumskip-method"></a>ICorDebugEnum::Skip 메서드
열거형에 커서를 앞으로 항목의 지정한 수 만큼 이동 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Skip (  
    [in] ULONG celt  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `celt`  
 [in] 커서를 앞으로 이동 하려는 항목의 수입니다.
