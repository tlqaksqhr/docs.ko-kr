# <a name="icordebugbreakpointactivate-method"></a>ICorDebugBreakpoint::Activate 메서드
이 활성 상태를 설정 `ICorDebugBreakpoint`합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Activate (  
    [in] BOOL bActive  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `bActive`  
 [in] 이 값을 설정 `true` 으로 활성화 상태를 지정 하려면 그렇지 않은 경우이 값을 설정 `false`합니다.
