# <a name="icordebugobjectenum-interface1"></a>ICorDebugObjectEnum Interface1

## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[Next 메서드](../../../../docs/framework/unmanaged-api/debugging/icordebugobjectenum-next-method.md)|개체의 지정된 된 숫자의 rva가 현재 위치부터 시작 하는 열거형에서 가져옵니다.|
