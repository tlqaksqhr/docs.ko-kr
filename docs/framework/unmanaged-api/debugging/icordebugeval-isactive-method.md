# <a name="icordebugevalisactive-method"></a>ICorDebugEval::IsActive 메서드
ICorDebugEval 개체가이 현재 실행 되 고 있는지 여부를 나타내는 값을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT IsActive (  
    [out] BOOL              *pbActive  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pbActive`  
 [out] 이 평가 활성 상태 인지 여부를 나타내는 값에 대 한 포인터입니다.
