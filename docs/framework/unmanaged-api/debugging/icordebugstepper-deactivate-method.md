이 ICorDebugStepper 받은 마지막 단계 명령 취소 하면 됩니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Deactivate ();  
```  
  
## <a name="remarks"></a>설명  
 가장 최근에 수신한 단계 명령이 취소 된 후에 새 단계별 실행 명령을 실행할 수 있습니다.
