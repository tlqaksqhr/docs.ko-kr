# <a name="couninitializecor-function"></a>CoUninitializeCor 함수
`CoUninitializeCor`는 사용되지 않습니다.  
  
## <a name="syntax"></a>구문  
  
```  
STDAPI_(void) CoUninitializeCor(void);  
```  
  
## <a name="remarks"></a>설명  
 공용 언어 런타임 프로세스에서 언로드할 수 없습니다. 런타임에 실행 중인 프로세스에서를 완전히 제거 하려면 해당 프로세스를 종료 해야 합니다.
