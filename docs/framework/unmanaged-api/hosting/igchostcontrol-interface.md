# <a name="igchostcontrol-interface"></a>IGCHostControl 인터페이스
가비지 수집기를 호스트에 가상 메모리의 한계를 변경 하도록 요청을 허용 하는 메서드를 제공 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[RequestVirtualMemLimit 메서드](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|호스트에 가상 메모리의 한계를 변경 하도록 요청 합니다.|
