# 사용자 지정 활동 디자이너 및 템플릿 사용
이 단원의 항목에서는 사용자 지정 활동 디자이너와 사용자 지정 활동 템플릿을 만드는 방법에 대해 설명합니다.  
  
## 단원 내용  
 [방법: 사용자 지정 활동 디자이너 만들기](../../../docs/framework/windows-workflow-foundation//how-to-create-a-custom-activity-designer.md)  
 워크플로에서 제공된 디자이너가 디자인 작업에 적합하지 않을 경우 사용자 지정 활동 디자이너를 만드는 방법에 대해 설명합니다.  
  
 [방법: 사용자 지정 활동 템플릿 만들기](../../../docs/framework/windows-workflow-foundation//how-to-create-a-custom-activity-template.md)  
 사용자가 각 활동을 개별적으로 만들고 해당 속성과 기타 설정을 수동으로 구성하지 않아도 되도록 사용자 지정 활동 템플릿을 사용하여 활동을 미리 구성하는 방법에 대해 설명합니다.  
  
 [ModelItem 편집 컨텍스트 사용](../../../docs/framework/windows-workflow-foundation//using-the-modelitem-editing-context.md)  
 ModelItem 편집 컨텍스트의 기능을 사용하여 디자이너가 호스트와 상호 작용할 수 있는 방법에 대해 설명합니다.  
  
 [사용자 지정 작업 속성을 디자이너 컨트롤에 바인딩](../../../docs/framework/windows-workflow-foundation//binding-a-custom-activity-property-to-a-designer-control.md)  
 디자이너에서 ListView 컨트롤을 활동 속성에 바인딩하는 방법에 대해 설명합니다.
