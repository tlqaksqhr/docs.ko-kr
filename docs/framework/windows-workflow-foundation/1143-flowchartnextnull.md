|||  
|-|-|  
|ID|1143|  
|키워드|WFActivities|

## <a name="description"></a>설명  
 Flowchart에서 다음 노드가 null임을 나타냅니다. Flowchart 실행이 종료됩니다.  
  
## <a name="message"></a>메시지  
 Flowchart '%1'/FlowStep - 다음 노드가 null입니다. Flowchart 실행이 종료됩니다.
