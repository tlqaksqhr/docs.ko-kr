# 부분 신뢰
[!INCLUDE[netfx35_long](../../../../includes/netfx35-long-md.md)]부터는 부분적으로 신뢰하는 호출자가 <xref:System.ServiceModel>, <xref:System.Runtime.Serialization> 및 <xref:System.ServiceModel.Web>에서 구현된 public 형식 및 메서드에 액세스할 수 있습니다.이 단원에서는 낮은 CAS\(코드 액세스 보안\) 권한으로 실행되는 응용 프로그램에 사용할 수 있는 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 기능의 제한된 하위 집합 및 부분 신뢰 응용 프로그램 내에서 [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)]의 사용을 지원하는 시나리오에 대해 설명합니다.
