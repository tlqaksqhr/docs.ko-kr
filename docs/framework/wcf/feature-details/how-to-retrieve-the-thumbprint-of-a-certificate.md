2.  <span data-ttu-id="bb03f-117">**콘솔 루트** 창의 왼쪽 창에서 **인증서(로컬 컴퓨터)** 를 클릭합니다.</span><span class="sxs-lookup"><span data-stu-id="bb03f-117">In the **Console Root** window's left pane, click **Certificates (Local Computer)**.</span></span>

8.  <span data-ttu-id="bb03f-124">필드 목록을 스크롤하여 **지문**을 클릭합니다.</span><span class="sxs-lookup"><span data-stu-id="bb03f-124">Scroll through the list of fields and click **Thumbprint**.</span></span>
