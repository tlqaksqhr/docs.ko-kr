## <a name="findresponse"></a><span data-ttu-id="00c82-140">FindResponse</span><span class="sxs-lookup"><span data-stu-id="00c82-140">FindResponse</span></span>  
 <span data-ttu-id="00c82-141"><xref:System.ServiceModel.Discovery.FindResponse>에는 네트워크에서 일치하는 서비스가 보낸 응답을 포함하는 <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> 컬렉션 속성이 있습니다.</span><span class="sxs-lookup"><span data-stu-id="00c82-141"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="00c82-142">응답하는 서비스가 없으면 컬렉션이 비어 있습니다.</span><span class="sxs-lookup"><span data-stu-id="00c82-142">If no services replied, the collection is empty.</span></span> <span data-ttu-id="00c82-143">하나 이상의 서비스가 응답하면 서비스 주소 및 계약을 비롯한 서비스에 대한 일부 추가 정보가 포함된 <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> 개체에 각 응답이 저장됩니다.</span><span class="sxs-lookup"><span data-stu-id="00c82-143">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>  
  
 <span data-ttu-id="00c82-144">다음 예제에서는 코드에서 찾기 작업을 수행하는 방법을 보여 줍니다.</span><span class="sxs-lookup"><span data-stu-id="00c82-144">The following example shows how to perform a find operation in code.</span></span>  
  
```  
// Create DiscoveryClient  
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());  
  
// Create FindCriteria  
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));  
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));  
findCriteria.Duration = TimeSpan.FromSeconds(10);   
  
// Find ICalculatorService endpoints              
FindResponse findResponse = discoveryClient.Find(findCriteria);  
  
Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)  
```
