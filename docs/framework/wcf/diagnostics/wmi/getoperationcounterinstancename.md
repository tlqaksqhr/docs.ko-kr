## <a name="syntax"></a>구문  
  
```  
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## <a name="parameters"></a>매개 변수  
  
### <a name="operation"></a>작업  
 [in] string

|---------|-----------------------------------|
