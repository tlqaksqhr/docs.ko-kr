# <a name="participantrecoverylogentrycreationfailure"></a>ParticipantRecoveryLogEntryCreationFailure
Id: 141  
  
 심각도: 오류  
  
 범주: TransactionBridge  
  
## <a name="description"></a>설명  
 이 이벤트는 참가자 인리스트먼트에 대해 복구 로그 항목을 생성할 수 없음을 나타냅니다. 트랜잭션이 중단됩니다. 이벤트는 트랜잭션 ID, 인리스트먼트 ID, 예외, 프로세스 이름 및 프로세스 ID를 나열합니다.
