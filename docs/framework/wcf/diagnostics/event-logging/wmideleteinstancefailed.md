# <a name="wmideleteinstancefailed"></a>WmiDeleteInstanceFailed
Id: 122  
  
 심각도: 오류  
  
 범주: ServiceModel

 이 이벤트는 WMI DeleteInstance 요청이 처리되지 않았음을 나타냅니다. 이 이벤트는 예외, 프로세스 이름 및 프로세스 ID를 표시합니다.
