# 4600 - MessageLogEventSizeExceeded
## 속성  
  
|||  
|-|-|  
|ID|4600|  
|키워드가|WCFMessageLogging|  
|수준|경고|  
|채널|Microsoft\-Windows\-응용 프로그램 서버\-응용 프로그램\/디버그|  
  
## 설명  
 이 이벤트는 ETW 이벤트 크기를 초과하여 메시지를 기록할 수 없을 때 내보내집니다.  
  
## 메시지  
 ETW 이벤트 크기를 초과하므로 메시지를 기록할 수 없습니다.
