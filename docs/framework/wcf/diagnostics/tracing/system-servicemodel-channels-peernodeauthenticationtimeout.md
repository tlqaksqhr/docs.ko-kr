# System.ServiceModel.Channels.PeerNodeAuthenticationTimeout
보안 핸드셰이크가 지정된 제한 시간 내에 성공하지 못했습니다.  
  
## 설명  
 이 추적은 보안 환경 연결을 설정하는 동안 발생합니다.  
  
## 문제 해결  
 모든 환경 연결에 적절한 보안 자격 증명이 있고 1분 내에 모든 메시지에 응답하는지 확인합니다.
