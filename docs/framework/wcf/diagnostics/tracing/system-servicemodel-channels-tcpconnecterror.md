# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
TCP 연결 작업을 수행할 수 없습니다.  
  
## <a name="description"></a>설명  
 이 경고 수준 추적은 TCP 끝점에 연결하지 못했음을 나타냅니다. 이 오류는 원격 끝점이 지정된 IP 주소 및 포트에서 응답하지 않는 경우 발생할 수 있습니다. 이후에 다른 유효한 IP 주소(예를 들어 IPv4 또는 IPv6 주소 또는 지정된 호스트 이름을 나타내는 다른 IP 주소)에 대한 연결을 시도하여 성공한 경우 이 추적은 무시될 수 있습니다. 이 추적 내의 예외는 오류에 대한 추가 정보를 나타낼 수 있습니다.
