# System.ServiceModel.Channels.HttpChannelResponseAborted
HTTP를 통해 응답 메시지를 보내지 못했습니다.  
  
## 설명  
 이 경고 수준 추적은 HTTP 채널 응답이 중단되었음을 나타냅니다.이 추적의 추가 정보는 Abort 메서드에 전달된 메시지나 이유를 표시합니다.
