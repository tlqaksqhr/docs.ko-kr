# <a name="call-duration"></a>Call Duration
카운터 이름: Call Duration  
  
## <a name="description"></a>설명  
 이 작업에 대한 평균 호출 기간입니다. 평균 기간은 (N1-N0)/(D1-D0) 공식을 사용해 계산됩니다.
