## <a name="in-this-section"></a>단원 내용

 이 샘플에서는 서비스 작업 중에 서비스 이벤트 감사를 활성화하기 위해 <xref:System.ServiceModel.Description.ServiceSecurityAuditBehavior>를 사용하는 방법을 보여 줍니다.  
  
 [멤버 자격 및 역할 공급자](../../../../docs/framework/wcf/samples/membership-and-role-provider.md)

 [서비스 작업에 대 한 액세스 승인](../../../../docs/framework/wcf/samples/authorizing-access-to-service-operations.md)

 [클라이언트 가장](../../../../docs/framework/wcf/samples/impersonating-the-client.md)
