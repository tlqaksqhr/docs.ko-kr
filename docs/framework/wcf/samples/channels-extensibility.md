# 채널 확장성
이 단원에는 사용자 지정 채널을 보여 주는 샘플이 포함되어 있습니다.  
  
## 단원 내용  
 [로컬 채널](../../../../docs/framework/wcf/samples/local-channel.md)  
 동일한 응용 프로그램 도메인 내에서 통신하는 데 사용되는 로컬 채널인 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 전송 채널을 보여 줍니다.  
  
 [신뢰할 수 있는 보안 프로필](../../../../docs/framework/wcf/samples/reliable-secure-profile.md)  
 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] 및 RSP\(Reliable Secure Profile\)를 구성하는 방법을 보여 줍니다.  
  
 [사용자 지정 채널 디스패처](../../../../docs/framework/wcf/samples/custom-channel-dispatcher.md)  
 <xref:System.ServiceModel.ServiceHostBase>를 직접 구현하여 사용자 지정 방식으로 채널 스택을 빌드하는 방법과 웹 호스트 환경에서 사용자 지정 채널 디스패처를 만드는 방법을 보여 줍니다.  
  
 [청크 채널](../../../../docs/framework/wcf/samples/chunking-channel.md)  
 [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)]를 사용하여 전송된 큰 메시지를 버퍼링하는 데 사용되는 메모리 양을 제한하는 방법을 보여 줍니다.  
  
 [HTTP 승인 채널](../../../../docs/framework/wcf/samples/http-acknowledgement-channel.md)  
 단방향 메시징 패턴을 변경하는 계층화된 채널을 보여 줍니다.  
  
 [HttpCookieSession](../../../../docs/framework/wcf/samples/httpcookiesession.md)  
 사용자 지정 프로토콜 채널을 빌드하여 세션 관리에 HTTP 쿠키를 사용하는 방법을 보여 줍니다.  
  
 [사용자 지정 메시지 인터셉터](../../../../docs/framework/wcf/samples/custom-message-interceptor.md)  
 런타임 스택의 특정 지점에서 들어오고 나가는 모든 메시지를 가로채기 위해 채널 팩터리 및 채널 수신기를 만드는 사용자 지정 바인딩 요소의 구현 방법을 보여 줍니다.
