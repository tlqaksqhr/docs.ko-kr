# <a name="how-to-create-a-new-setting-at-design-time"></a>방법: 디자인 타임에 새 설정 만들기
설정 디자이너를 사용 하 여 디자인 타임에 새 설정을 만들 수 있습니다. 설정 디자이너는 새 설정을 만들고 이러한 설정에 대 한 속성을 지정할 수 있도록 표 스타일 인터페이스입니다. 이름, 값, 유형 및 새 설정에 대 한 범위를 지정 해야 합니다. 설정을 만든 후에 코드에서 액세스할 수 있습니다.  
  
### <a name="to-create-a-new-setting-at-design-time-in-c"></a>C#에서 디자인 타임에 새 설정을 만들려면  
  
1.  **솔루션 탐색기**를 확장 하 고는 **속성** 프로젝트의 노드.  
  
2.  새 설정을 추가 하려는.settings 파일을 두 번 클릭 합니다. 이 파일에 대 한 기본 이름은 Settings.settings입니다.  
  
3.  설정 디자이너에서 이름, 값, 형식 및 사용자 설정에 대 한 범위를 설정 합니다. 각 행에는 단일 설정을 나타냅니다.  
  
### <a name="to-create-a-new-setting-at-design-time-in-visual-basic"></a>Visual Basic의 디자인 타임에 새 설정을 만들려면  
  
1.  **솔루션 탐색기**프로젝트 노드를 마우스 오른쪽 단추로 클릭 하 고 선택 **속성**합니다.  
  
2.  에 **속성** 선택 페이지는 **설정을** 탭 합니다.  
  
3.  설정 디자이너에서 이름, 값, 형식 및 사용자 설정에 대 한 범위를 설정 합니다. 각 행에는 단일 설정을 나타냅니다.
