# <a name="how-to-change-the-value-of-an-existing-setting-at-design-time"></a>방법: 디자인 타임에 기존 설정 값 변경
프로젝트에 기존 설정의 값을 편집 하려면 Visual Studio를 사용할 수 있습니다.  
  
### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-c"></a>C#에서 디자인 타임에 기존 설정의 값을 변경 하려면  
  
1.  **솔루션 탐색기**를 확장 하 고는 **속성** 프로젝트의 노드.  
  
2.  새 설정을 추가 하려는.settings 파일을 두 번 클릭 합니다. 이 파일에 대 한 기본 이름은 Settings.settings입니다.  
  
3.  설정 디자이너에서 값을 변경 하 고 값 열에 새 값을 입력 하 고 설정을 찾습니다.  
  
### <a name="to-change-the-value-of-an-existing-setting-at-design-time-in-visual-basic"></a>Visual Basic에서 디자인 타임에 기존 설정의 값을 변경 하려면  
  
1.  **솔루션 탐색기**프로젝트 노드를 마우스 오른쪽 단추로 클릭 하 고 선택 **속성**합니다.  
  
2.  에 **속성** 선택 페이지는 **설정을** 탭 합니다.  
  
3.  설정 디자이너에서 값을 변경 하 고 값 열에 새 값을 입력 하 고 설정을 찾습니다.
