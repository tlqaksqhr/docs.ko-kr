# <a name="picturebox-control-overview-windows-forms"></a>PictureBox 컨트롤 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.PictureBox> 컨트롤은 그래픽 비트맵, GIF, JPEG, 메타 파일 또는 아이콘으로 표시 하는 데 사용 합니다.  
  
## <a name="key-properties-and-methods"></a>키 속성 및 메서드  
 표시 되는 그림에 의해 결정 됩니다는 <xref:System.Windows.Forms.PictureBox.Image%2A> 런타임 또는 디자인 타임에 설정 될 수 있는 속성입니다. 이미지를 설정 하 여 지정할 수 있습니다는 <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> 속성 동기적으로 처리를 사용 하 여 이미지를 로드 하 고는 <xref:System.Windows.Forms.PictureBox.Load%2A> 메서드 또는 비동기적으로 사용 하는 <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> 메서드. <xref:System.Windows.Forms.PictureBox.SizeMode%2A> 속성 이미지 및 제어를 서로 맞추는 방법을 제어 합니다. 자세한 내용은 참조 [하는 방법: 크기 또는 런타임에 그림의 위치를 수정](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md)합니다.
