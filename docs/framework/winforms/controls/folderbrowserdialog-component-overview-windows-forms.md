# <a name="folderbrowserdialog-component-overview-windows-forms"></a>FolderBrowserDialog 구성 요소 개요(Windows Forms)
Windows Forms <xref:System.Windows.Forms.FolderBrowserDialog> 구성 요소는 폴더를 선택 하 고 검색에 사용 되는 모달 대화 상자. 내에서 새 폴더를 만들 수도 있습니다는 <xref:System.Windows.Forms.FolderBrowserDialog> 구성 요소입니다.  
  
> [!NOTE]
>  폴더 대신 파일을 선택 하려면 사용 하 여는 [OpenFileDialog](../../../../docs/framework/winforms/controls/openfiledialog-component-windows-forms.md) 구성 요소입니다.  
  
 <xref:System.Windows.Forms.FolderBrowserDialog> 구성 요소가 사용 하 여 런타임에 표시 되는 <xref:System.Windows.Forms.CommonDialog.ShowDialog%2A> 메서드. 설정의 <xref:System.Windows.Forms.FolderBrowserDialog.RootFolder%2A> 최상위 폴더 및 대화 상자의 트리 뷰 내에서 나타나는 모든 하위 폴더를 확인 하는 속성입니다. 대화 상자가 표시 되 면 사용할 수 있습니다는 <xref:System.Windows.Forms.FolderBrowserDialog.SelectedPath%2A> 선택 된 폴더의 경로 가져올 속성입니다.  
  
 폼에 추가 될 때의 <xref:System.Windows.Forms.FolderBrowserDialog> 구성 요소는 Windows Forms 디자이너 아래쪽에서 트레이에 표시 됩니다.
