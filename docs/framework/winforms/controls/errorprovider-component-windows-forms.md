# <a name="errorprovider-component-windows-forms"></a>ErrorProvider 구성 요소(Windows Forms)
Windows Forms `ErrorProvider` 구성 요소는 문제가 있다고 자연스럽 방식으로 사용자에 게 표시 하는 데 사용 됩니다. 일반적으로 폼에서 사용자 입력 유효성 검사 또는 데이터 집합의 오류 표시와 함께에서 사용 됩니다.

 이 구성 요소의 정의와 주요 기능 및 속성에 설명 합니다.  
  
 [방법: Windows Forms ErrorProvider 구성 요소를 사용하여 폼 유효성에 대한 오류 아이콘 표시](../../../../docs/framework/winforms/controls/display-error-icons-for-form-validation-with-wf-errorprovider.md)  
 오류 공급자 구성 요소를 사용 하 여 사용자 입력 유효성 검사에 대 한 지침을 제공 합니다.  
  
 [방법: Windows Forms ErrorProvider 구성 요소를 사용하여 데이터 집합에 있는 오류 보기](../../../../docs/framework/winforms/controls/view-errors-within-a-dataset-with-wf-errorprovider-component.md)  
 데이터 오류를 표시 하려면 오류 공급자 구성 요소를 사용 하기 위한 지침을 제공 합니다.
