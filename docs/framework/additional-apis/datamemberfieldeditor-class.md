# <a name="datamemberfieldeditor-class"></a>DataMemberFieldEditor 클래스

모든 속성을 나열 하 여 데이터 바인딩된 개체의 속성을 편집 하기 위한 사용자 인터페이스를 제공는 `DataSource` 에서 값의 소스를 선택 하는 개체입니다.  
  
## <a name="syntax"></a>구문
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> `DataMemberFieldEditor` 클래스는 내부 전용 이며 코드에서 직접 사용할 업그레이드용은 아닙니다.
> 
> Microsoft은 프로덕션 응용 프로그램의 어떤 상황에서이 클래스의 사용을 지원 하지 않습니다.
