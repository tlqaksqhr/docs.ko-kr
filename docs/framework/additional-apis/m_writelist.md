# <a name="connectionmwritelist-field"></a>Connection.m\_WriteList 필드

`Connection.m_WriteList` 이 <xref:System.Collections.ArrayList> 의 <xref:System.Net.HttpWebRequest> HTTP를 통해 전송 되도록 하는 개체입니다.

## <a name="syntax"></a>구문
  
```csharp  
private ArrayList m_WriteList
```

> [!WARNING]
> `Connection.m_WriteList` 필드는 전용 이며 코드에서 직접 사용할 업그레이드용은 아닙니다.
> 
> Microsoft은 프로덕션 응용 프로그램의 어떤 상황에서이 필드의 사용을 지원 하지 않습니다.

## <a name="requirements"></a>요구 사항

**Namespace:** <xref:System.Net>

**어셈블리:** 시스템 (System.dll)

**.NET framework 버전:** 2.0부터 사용 가능 합니다.
