 <span data-ttu-id="d31c5-292">다음 예제에서는 `AppContextSwitchOverrides` 단일 응용 프로그램 호환성 스위치를 정의 하는 요소 `Switch.System.Globalization.NoAsyncCurrentCulture`, 비동기 메서드 호출에 스레드를 넘어서 전달에서 culture에 맞지 않는 합니다.</span><span class="sxs-lookup"><span data-stu-id="d31c5-292">The following example uses the `AppContextSwitchOverrides` element to define a single application  compatibility switch, `Switch.System.Globalization.NoAsyncCurrentCulture`, that prevents culture from flowing across threads in asynchronous method calls.</span></span>

 <span data-ttu-id="d31c5-293">다음 예제에서는 `AppContextSwitchOverrides` 두 응용 프로그램 호환성 스위치를 정의 하는 요소 `Switch.System.Globalization.NoAsyncCurrentCulture` 및 `Switch.System.IO.BlockLongPaths`합니다.</span><span class="sxs-lookup"><span data-stu-id="d31c5-293">The following example uses the `AppContextSwitchOverrides` element to define two application  compatibility switches, `Switch.System.Globalization.NoAsyncCurrentCulture` and `Switch.System.IO.BlockLongPaths`.</span></span> <span data-ttu-id="d31c5-294">Note 세미콜론 두 개의 이름/값 쌍을 구분 합니다.</span><span class="sxs-lookup"><span data-stu-id="d31c5-294">Note that a semicolon separates the two name/value pairs.</span></span>
