 <span data-ttu-id="fdda8-159">DLR은 동적 언어 지원을 개선하기 위해 서비스 집합을 CLR에 추가합니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-159">The DLR adds a set of services to the CLR for better supporting dynamic languages.</span></span> <span data-ttu-id="fdda8-160">이러한 서비스에는 다음 항목이 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-160">These services include the following:</span></span>  
  
-   <span data-ttu-id="fdda8-161">식 트리.</span><span class="sxs-lookup"><span data-stu-id="fdda8-161">Expression trees.</span></span> <span data-ttu-id="fdda8-162">DLR은 식 트리를 사용하여 언어 의미 체계를 나타냅니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-162">The DLR uses expression trees to represent language semantics.</span></span> <span data-ttu-id="fdda8-163">이 목적으로 DLR은 제어 흐름, 할당 및 기타 언어 모델링 노드를 포함하도록 LINQ 식 트리를 확장했습니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-163">For this purpose, the DLR has extended LINQ expression trees to include control flow, assignment, and other language-modeling nodes.</span></span> <span data-ttu-id="fdda8-164">자세한 내용은 [식 트리](http://msdn.microsoft.com/library/fb1d3ed8-d5b0-4211-a71f-dd271529294b)를 참조하세요.</span><span class="sxs-lookup"><span data-stu-id="fdda8-164">For more information, see [Expression Trees](http://msdn.microsoft.com/library/fb1d3ed8-d5b0-4211-a71f-dd271529294b).</span></span>  
  
-   <span data-ttu-id="fdda8-165">호출 사이트 캐싱.</span><span class="sxs-lookup"><span data-stu-id="fdda8-165">Call site caching.</span></span> <span data-ttu-id="fdda8-166">*동적 호출 사이트*는 코드에서 동적 개체에 대해 `a + b` 또는 `a.b()` 같은 작업을 수행하는 위치입니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-166">A *dynamic call site* is a place in the code where you perform an operation like `a + b` or `a.b()` on dynamic objects.</span></span> <span data-ttu-id="fdda8-167">DLR은 `a` 및 `b`의 특징(대개 이러한 개체의 형식)과 작업 정보를 캐시합니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-167">The DLR caches the characteristics of `a` and `b` (usually the types of these objects) and information about the operation.</span></span> <span data-ttu-id="fdda8-168">해당 작업이 이전에 수행된 적이 있으면 DLR은 빠른 디스패치를 위해 캐시에서 모든 필요한 정보를 검색합니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-168">If such an operation has been performed previously, the DLR retrieves all the necessary information from the cache for fast dispatch.</span></span>  
  
-   <span data-ttu-id="fdda8-169">동적 개체 상호 운용성.</span><span class="sxs-lookup"><span data-stu-id="fdda8-169">Dynamic object interoperability.</span></span> <span data-ttu-id="fdda8-170">DLR은 동적 개체와 개체를 나타내고 언어 구현자 및 동적 라이브러리 작성자가 사용할 수 있는 클래스 및 인터페이스 집합을 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-170">The DLR provides a set of classes and interfaces that represent dynamic objects and operations and can be used by language implementers and authors of dynamic libraries.</span></span> <span data-ttu-id="fdda8-171">이러한 클래스 및 인터페이스에는 <xref:System.Dynamic.IDynamicMetaObjectProvider>, <xref:System.Dynamic.DynamicMetaObject>, <xref:System.Dynamic.DynamicObject> 및 <xref:System.Dynamic.ExpandoObject>가 포함됩니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-171">These classes and interfaces include <xref:System.Dynamic.IDynamicMetaObjectProvider>, <xref:System.Dynamic.DynamicMetaObject>, <xref:System.Dynamic.DynamicObject>, and <xref:System.Dynamic.ExpandoObject>.</span></span>  
  
 <span data-ttu-id="fdda8-172">DLR은 호출 사이트에서 바인더를 사용하여 .NET Framework뿐 아니라 Silverlight 및 COM이 포함된 다른 인프라 및 서비스와 통신합니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-172">The DLR uses binders in call sites to communicate not only with the .NET Framework, but with other infrastructures and services, including Silverlight and COM.</span></span> <span data-ttu-id="fdda8-173">바인더는 언어의 의미 체계를 캡슐화하고 호출 사이트에서 식 트리를 사용하여 작업을 수행하는 방법을 지정합니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-173">Binders encapsulate a language's semantics and specify how to perform operations in a call site by using expression trees.</span></span> <span data-ttu-id="fdda8-174">이를 통해 동적 언어와 정적 형식의 언어가 DLR을 사용하여 라이브러리를 공유하고 DLR이 지원하는 모든 기술에 액세스할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="fdda8-174">This enables dynamic and statically typed languages that use the DLR to share libraries and gain access to all the technologies that the DLR supports.</span></span>
