
> [!NOTE]
> 이 문서의 C# 예제는 [Try.NET](https://try.dot.net) 인라인 코드 러너 및 놀이터에서 실행됩니다. 대화형 창에서 예제를 실행하려면 **실행** 버튼을 선택합니다. 코드를 실행하면 **실행**을 다시 선택하여 코드를 수정하고 수정된 코드를 실행할 수 있습니다. 수정된 코드는 대화형 창에서 실행되거나, 컴파일이 실패하면 대화형 창에 모든 C# 컴파일러 오류 메시지가 표시됩니다. 
>  
> [Try.NET](https://try.dot.net) 인라인 코드 실행기 및 플레이그라운드의 [현재 문화권](xref:System.Globalization.CultureInfo.CurrentCulture)은 [고정 문화권](xref:System.Globalization.CultureInfo.InvariantCulture)입니다. 따라서 인라인 코드 실행기에서 생성된 출력은 기본 문화권을 현재 문화권으로 사용하는 예제에서 표시된 출력과 다릅니다.
